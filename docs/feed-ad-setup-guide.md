# 피드 광고 5종 — Meta 광고 관리자 실제 화면 기준 입력 가이드

> 작성일: 2026-04-10
> 목적: Meta 광고 관리자의 실제 화면 순서대로, 이 문서만 보고 복붙하면 광고 5개를 완성할 수 있도록 작성
> 캠페인: "인테리어_소재선별_Phase1"
> 광고 세트: "피드_5종테스트"
> 계정: Facebook 페이지 "모아홈테리어" / Instagram "moahome_interior"

## 광고 화면 구조 (위에서 아래로)

Meta 광고 관리자의 광고 편집 화면은 위에서 아래로 이렇게 구성되어 있습니다:

```
① 광고 이름
② 파트너십 광고
③ 대표 계정 (Facebook 페이지 / Instagram 프로필)
④ 광고 설정 (형식 선택)
⑤ 랜딩 페이지 + 웹사이트 URL
⑥ 광고 크리에이티브 ← 이미지 업로드 + 텍스트 입력이 여기!
⑦ 크리에이티브 테스트
⑧ 언어
⑨ 추적 (UTM 매개변수)
```

## 첫 번째 광고 (F1) — 화면 순서대로 완전 가이드

### ① 광고 이름

```
F1_궁금증_1600만원
```

### ② 파트너십 광고

```
해제 (건드리지 않기)
```

### ③ 대표 계정

| 항목 | 값 |
|------|-----|
| Facebook 페이지 | **모아홈테리어** (이미 선택되어 있음) |
| Instagram 프로필 | **moahome_interior** (이미 선택되어 있음) |

> 건드리지 않으면 됩니다.

### ④ 광고 설정

```
광고 만들기 (이미 선택됨)
```

아래에 보이는 항목들:

| 항목 | 선택 |
|------|------|
| 크리에이티브 소스 | **수동 업로드** (이미 선택됨) |
| 형식 | **단일 이미지 또는 동영상** (이미 선택됨) |
| 여러 광고주의 광고 | 체크되어 있으면 **그대로 두기** |

### ⑤ 랜딩 페이지

| 항목 | 입력값 |
|------|-------|
| 랜딩 페이지 유형 | **웹사이트** (이미 선택됨) |
| 웹사이트 URL | `https://www.moahome.kr/` |
| 표시 링크 | `moahome.kr` (선택사항 — 빈칸이어도 OK) |
| 브라우저 추가 기능 | **없음** (이미 선택됨) |

### ⑥ 광고 크리에이티브 (핵심!)

> 여기가 이미지 업로드 + 텍스트 입력하는 곳입니다.

**현재 화면에 이렇게 보일 겁니다:**

```
광고 크리에이티브
광고 문구, 미디어 및 개선 사항을 선택하고 최적화합니다.

[크리에이티브 설정 ▼]

ⓘ 이 광고에 게재할 이미지를 지정해주세요.
```

#### Step A: "크리에이티브 설정" 드롭다운 클릭

```
[크리에이티브 설정 ▼] ← 이 버튼을 클릭!
```

클릭하면 아래에 다음이 펼쳐집니다:

```
미디어
┌──────────────────────────────────────┐
│  [+ 미디어 추가]  ← 이 버튼 클릭!    │
│                                       │
│  또는 이미지를 여기에 드래그           │
└──────────────────────────────────────┘

기본 텍스트
┌──────────────────────────────────────┐
│  텍스트 입력란                        │
└──────────────────────────────────────┘

헤드라인
┌──────────────────────────────────────┐
│  텍스트 입력란                        │
└──────────────────────────────────────┘

설명
┌──────────────────────────────────────┐
│  텍스트 입력란                        │
└──────────────────────────────────────┘

행동 유도 (CTA 버튼)
[자세히 알아보기 ▼]
```

#### Step B: 이미지 업로드

```
1. [+ 미디어 추가] 클릭
2. "업로드" 탭 선택
3. PC에서 feed1.png 파일 선택
4. 업로드 완료 → 이미지 미리보기가 나타남
5. 우측 "광고 미리 보기"에서 Instagram 피드에 이미지가 보이는지 확인
```

#### Step C: 텍스트 입력 — F1용

**기본 텍스트** (본문 — 가장 큰 텍스트 박스):

```
같은 평수인데 견적이 천만원 넘게 차이나는 거 아셨나요?

비교 한 번이면 수백만원 아낄 수 있습니다.
AI가 완성된 인테리어 모습까지 무료로 미리 보여드려요.

30초 무료 견적 비교 시작하기
```

**헤드라인** (이미지 바로 아래에 표시되는 짧은 제목):

```
30초면 내 견적 확인 끝
```

**설명** (헤드라인 아래 작은 글씨):

```
무료 인테리어 비교견적 서비스
```

**행동 유도 (CTA 버튼):**

```
드롭다운에서 "자세히 알아보기" 선택
```

### ⑦ 크리에이티브 테스트

```
건드리지 않기 (테스트 설정 버튼 누르지 않기)
```

### ⑧ 언어

```
해제됨 (건드리지 않기)
```

### ⑨ 추적

하단으로 스크롤하면 "추적" 섹션이 보입니다.

**URL 매개변수** 입력란에 아래를 복붙:

```
utm_source=meta&utm_medium=ig_feed&utm_campaign=phase1&utm_content=F1_curiosity
```

> "URL 매개변수 만들기" 링크가 있으면 클릭해서 빌더를 쓸 수도 있고,
> 그냥 입력란에 위 텍스트를 직접 붙여넣어도 됩니다.

### ⑩ 확인 + 다음 광고로

```
1. 우측 "광고 미리 보기"에서 Instagram 피드 탭 확인
   → 이미지 + "더 알아보기" 버튼이 보이면 OK
2. 우측 상단의 "변경 사항 확인" 경고는 무시 (Audience Network 관련 — 우리와 무관)
```

## 최종 확인 체크리스트

5개 광고를 모두 등록한 후, **"동의하고 게시"** 버튼 누르기 전에 확인:

```
□ 광고 5개가 전부 같은 광고 세트("피드_5종테스트") 안에 있는지
  → 좌측 패널에서 광고 세트 아래 광고 5개가 보여야 함

□ 각 광고의 이미지가 올바른 파일인지
  → 우측 "광고 미리 보기"에서 Instagram 피드 탭 확인
  → feed1~5 이미지가 각각 맞는지 육안 대조

□ 웹사이트 URL 확인
  → F1, F2, F3: https://www.moahome.kr/
  → F4, F5: https://www.moahome.kr/form

□ UTM 매개변수가 각각 다른지
  → F1_curiosity / F2_hidden / F3_gap / F4_shock / F5_compare

□ CTA 버튼이 전부 "자세히 알아보기"인지

□ 기본 텍스트에 오타 없는지

□ "변경 사항 확인" 경고
  → "Audience Network에 뉴스피드가 필요함" → 무시해도 됨
  → 그 외 빨간 에러가 있으면 해결 필요
```

모두 확인했으면 → 하단 우측 **[동의하고 게시]** 파란 버튼 클릭!

## 게시 후 예상 타임라인

```
게시 직후      → Meta 광고 검토 시작 (자동)
30분~24시간    → 검토 완료 → 광고 노출 시작
               (검토 중이면 "검토 중" 상태로 표시됨 — 기다리면 됨)
1일차          → 데이터 축적 시작 (아직 판단 불가)
2일차          → 소재별 노출수 확인 (최소 500 이상인지)
3일차          → CTR 0.3% 미만 소재 있으면 끄기
4일차 (종료)   → 최종 CTR 순위 정렬 → 상위 4종 선정
```

## 자주 겪는 문제

### "이 광고에 게재할 이미지를 지정해주세요" 경고가 계속 뜸

→ **"크리에이티브 설정" 드롭다운을 클릭**해야 이미지 업로드 영역이 나타납니다.
드롭다운이 접혀 있으면 이미지를 올릴 곳이 안 보입니다.

### "Audience Network에 뉴스피드가 필요함" 경고

→ 우리는 Instagram 피드만 선택했으므로 **무시**. 게시에 영향 없음.

### 이미지 업로드 후 텍스트 입력란이 안 보임

→ "크리에이티브 설정" 드롭다운 아래로 스크롤.
→ 이미지 업로드 영역 아래에 기본 텍스트, 헤드라인, 설명 입력란이 있음.

### 광고 복제 시 이미지가 그대로 딸려옴

→ 정상입니다. 복제 후 이미지를 삭제(X 버튼)하고 새 이미지를 업로드하면 됩니다.
