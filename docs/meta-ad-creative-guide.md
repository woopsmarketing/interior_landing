# Meta(Instagram) 광고 소재 가이드 — 인테리어 견적 비교 플랫폼

> 작성일: 2026-03-18
> 목적: 월 50만원 테스트 예산으로 최대 전환을 뽑아내기 위한 광고 소재 상세 기획

---

## 1. 광고 세트 구성 전략

| 구분 | 세트 A | 세트 B |
|------|--------|--------|
| **포맷** | 카루셀 4장 | 싱글 이미지 1장 |
| **목적** | 스토리텔링으로 욕망+불안+해결 순서 전달 | 피드에서 즉시 클릭 유도 |
| **일 예산** | 8,500원 | 8,500원 |
| **운영** | 2주 A/B → 성과 좋은 쪽에 집중 | |

## 2. 세트 A — 카루셀 4장 상세

### 전체 스토리 흐름

```
1장: 공감 (이 집 낡았다 → 나도 저런데?)
  ↓ 넘김
2장: 욕망 (같은 집이 이렇게 바뀐다고?)
  ↓ 넘김
3장: 해결책 (AI 미리보기 + 플랫폼 소개)
  ↓ 넘김
4장: 행동 (지금 무료로 시작)
```

유저가 넘길 때마다 감정 강도가 올라가는 구조입니다.

### 카루셀 1장: Before — "공감 + 시선 멈춤"

#### 레이아웃

```
┌─────────────────────────────┐
│  (상단 여백 15%)             │
│                             │
│    ┌─────────────────┐      │
│    │                 │      │
│    │   낡은 거실      │      │
│    │   사진 전체      │      │
│    │   (이미지 80%)   │      │
│    │                 │      │
│    └─────────────────┘      │
│                             │
│  ┌───────────────────────┐  │
│  │ "솔직히,              │  │
│  │  이 집 바꾸는 데      │  │
│  │  얼마면 될까?"        │  │
│  └───────────────────────┘  │
│              (하단 텍스트)    │
└─────────────────────────────┘

사이즈: 1080 x 1080px
```

#### 이미지 생성 프롬프트 (ChatGPT)

```
오래되고 낡은 한국 아파트 25평 거실의 사실적인 사진.

- 누렇게 변색된 벽지, 군데군데 들뜬 자국
- 긁히고 색이 바랜 장판 바닥
- 오래된 형광등 조명 (차갑고 어두운 빛)
- 낡은 TV 장식장, 정리되지 않은 소품
- 답답하고 좁아 보이는 분위기
- 사실적인 사진 스타일, 와이드 정면 구도
- 정사각형 1:1 비율, 고해상도
```

#### Canva 텍스트 가이드

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 상단 태그 | `솔직히,` | 흰색, 16px, 반투명 검정 배경 태그 |
| 메인 카피 | `이 집 바꾸는 데 얼마면 될까?` | 흰색, Bold 28~32px, 텍스트 그림자 |
| 위치 | 이미지 하단 1/4 영역 | 하단 그라데이션 오버레이(검정→투명) |

#### 디자인 핵심 포인트
- 텍스트는 이미지 위에 직접 올려야 피드에서 자연스러움
- 하단에 검정→투명 그라데이션 깔면 텍스트 가독성 확보
- "솔직히," 라는 단어가 대화체라서 스크롤 멈춤 효과
- 이미지가 충분히 낡아 보여야 2장과 갭이 극대화됨

```
한국 아파트 25평 거실이 모던 미니멀 스타일로
완벽하게 리모델링된 사실적인 사진.

- 깨끗한 화이트 벽면
- 원목 헤링본 패턴 바닥
- 따뜻한 간접조명 + 큰 창으로 자연광

- 원목 테이블, 미니멀한 소품 배치
- 카페처럼 따뜻하고 세련된 분위기
- 인테리어 매거진 촬영 스타일, 와이드 정면 구도
- 정사각형 1:1 비율, 고해상도
```

#### Canva 텍스트 가이드

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 메인 카피 | `같은 집입니다.` | 흰색, Bold 32px, 텍스트 그림자 |
| 서브 카피 | `그런데 업체마다 견적이` | 흰색, Medium 18px |
| 강조 숫자 | `1,000만원 이상 차이납니다` | 오렌지(#FF6B00), Bold 22px |
| 위치 | 하단 1/3 영역 | 하단 그라데이션 오버레이 |

#### 디자인 핵심 포인트
- 1장과 **같은 구도**(정면 와이드)여야 "같은 집" 충격이 극대화
- "같은 집입니다" 한 마디로 호기심 → 욕망 전환
- 금액("1,000만원")을 오렌지색으로 강조해서 시선 집중
- 여기서 "왜 차이나지?" 궁금증 → 3장으로 넘기게 만듦

### 카루셀 3장: 플랫폼 소개 + AI 기능 — "해결책"

#### 레이아웃

```
┌──────────────────────────────────┐
│  배경: 크림색 (#FFF9F5)           │
│                                  │
│  ┌──────────────────────────┐    │
│  │  (상단 헤드라인)           │    │
│  │  "비교하면 답이 보입니다"   │    │
│  └──────────────────────────┘    │
│                                  │
│  ┌────────┐    ┌────────┐        │
│  │ Before │ →  │ After  │        │
│  │ 작은   │    │ AI생성  │        │
│  │ 사진   │    │ 사진   │        │
│  └────────┘    └────────┘        │
│  (AI 인테리어 미리보기 예시)       │
│                                  │
│  ─────────────────────────────   │
│                                  │
│  ✓ 100개+ 검증 업체 견적 비교     │
│  ✓ AI가 완성 모습 미리 보여줌     │
│  ✓ 2~3일 내 견적 답변             │
│  ✓ 100% 무료, 계약 강요 없음      │
│                                  │
│  (하단 작게)                      │
│  "사진 한 장이면 충분합니다"       │
└──────────────────────────────────┘

사이즈: 1080 x 1080px
```

#### 이미지 생성 프롬프트 (ChatGPT)

3장은 2가지 이미지가 필요합니다:

**Before 썸네일 (작게 쓸 용도)**
→ 1장에서 생성한 낡은 거실 이미지를 그대로 축소 사용

**After 썸네일 (AI 생성 결과 느낌)**
→ 2장에서 생성한 완성 거실 이미지를 그대로 축소 사용

또는 새로 생성하려면:
```
스마트폰 화면에 보이는 인테리어 비교 UI 목업.
왼쪽에 낡은 거실 사진, 오른쪽에 AI가 변환한
모던 인테리어 완성 사진이 나란히 배치.
사이에 화살표(→) 아이콘.
크림색(#FFF9F5) 배경, 깔끔한 디자인.
정사각형 1:1 비율.
```

#### Canva 제작 가이드

이 장은 **Canva에서 직접 조립**합니다:

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 단색 크림 `#FFF9F5` | |
| 헤드라인 | `비교하면 답이 보입니다` | 다크그레이(#1A1A1A), Bold 28px |
| Before/After | 1장·2장 이미지를 작게 (각 400x300px) | 둥근 모서리, 그림자, 사이에 → 화살표 |
| 캡션 | `AI 인테리어 미리보기` | 오렌지(#FF6B00), Medium 14px |
| 구분선 | 얇은 회색 라인 | |
| 체크리스트 | 4개 항목 (위 레이아웃 참조) | 다크그레이, Regular 16px, ✓는 오렌지 |
| 하단 문구 | `사진 한 장이면 충분합니다` | 그레이(#888), 14px |

#### 디자인 핵심 포인트
- 이 장은 "정보 전달" 목적이라 배경을 깔끔하게 (사진 X, 크림 배경)
- Before→After 작은 비교를 넣어서 "AI가 이런 걸 해준다"는 것을 시각적으로 전달
- 체크리스트 4개는 모두 고객 이익 중심 ("무료", "강요 없음", "빠른 답변")
- 텍스트 비율 20% 이하 유지하면서 핵심만

```

│  └──────────────────────────┘    │
│                                  │
│  ─────────── or ──────────────   │

│  │  ① 3분 입력이면 끝        │    │

│  └──────────────────────────┘    │
│                                  │
│  ┌──────────────────────────┐    │
│  │                          │    │
│  │   ▶ 무료 견적 비교하기     │    │
│  │                          │    │
│  └──────────────────────────┘    │
│  (CTA 버튼 모양)                  │
│                                  │

│  미리보기를 무료로 받아보세요       │
└──────────────────────────────────┘

사이즈: 1080 x 1080px
```

#### Canva 제작 가이드 (이미지 생성 불필요 — 텍스트 카드)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 오렌지 그라데이션 `#FF8C42` → `#FF6B00` | 좌상→우하 대각선 |
| 메인 카피 | `비교 없이 계약하면`<br>`평균 수백만원 더 냅니다` | 흰색, Bold 30px |
| 구분선 | 흰색 얇은 라인 | 반투명 50% |
| 특징 3개 | ① 3분 입력이면 끝<br>② 원하는 업체만 직접 선택<br>③ 완전 무료, 부담 제로 | 흰색, Medium 18px |
| CTA 버튼 | `무료 견적 비교하기` | 흰색 배경, 오렌지 텍스트, Bold 20px, 둥근 모서리 |
| 하단 문구 | `지금 신청하면 AI 인테리어 미리보기를 무료로 받아보세요` | 흰색 80%, 14px |

#### 디자인 핵심 포인트
- **오렌지 배경**으로 앞 3장과 확연히 다른 느낌 → "마지막 장" 느낌 확실
- 이 장의 역할은 오직 **"클릭시키기"**
- CTA 버튼을 실제 버튼처럼 디자인 (그림자 + 둥근 모서리)
- "수백만원 더 냅니다" → 손실 회피 심리 자극 (얻는 것보다 잃는 것에 2배 반응)
- 하단 "AI 미리보기 무료" → 추가 동기 부여

## 4. 텍스트 오버레이 공통 규칙

### 폰트

| 용도 | 추천 폰트 | Canva 검색어 |
|------|----------|-------------|
| 헤드라인 | Pretendard Bold | `프리텐다드` 또는 `Noto Sans KR Bold` |
| 본문 | Pretendard Medium | `프리텐다드` 또는 `Noto Sans KR Medium` |
| 숫자 강조 | Pretendard ExtraBold | 같은 폰트 굵기만 변경 |

### 색상 팔레트

| 용도 | 색상 | HEX |
|------|------|-----|
| 메인 텍스트 (밝은 배경) | 다크 그레이 | `#1A1A1A` |
| 메인 텍스트 (어두운 배경/사진 위) | 화이트 | `#FFFFFF` |
| 강조 (금액, CTA, 포인트) | 오렌지 | `#FF6B00` |
| 서브 텍스트 | 미디엄 그레이 | `#888888` |
| 배경 (텍스트 카드) | 크림 | `#FFF9F5` |
| CTA 배경 그라데이션 | 오렌지 | `#FF8C42` → `#FF6B00` |

### 텍스트 비율 규칙 (메타 정책)

```
이미지 대비 텍스트 면적: 20% 이하 (필수)
- 초과 시: 광고 도달 감소 또는 비승인
- Canva에서 텍스트 영역이 이미지의 1/5 이하인지 확인
- 의심되면 Meta 텍스트 오버레이 도구로 체크
```

## 5. 광고 캠페인 설정 가이드

### 캠페인 구조

```
캠페인: "인테리어 견적 비교 — 테스트"
├── 광고 세트 A: 카루셀 (일 8,500원)
│   ├── 타겟: 25~50세, 인테리어/부동산/이사 관심
│   └── 노출: 인스타그램 피드 + 스토리
└── 광고 세트 B: 싱글 이미지 (일 8,500원)
    ├── 타겟: 동일
    └── 노출: 인스타그램 피드 + 스토리
```

### 광고 관리자 설정값

| 항목 | 설정값 |
|------|--------|
| 캠페인 목표 | 전환 (웹사이트) 또는 트래픽 |
| 전환 이벤트 | Lead (Pixel 설치 후) |
| 일 예산 | 8,500원 × 2세트 |
| 노출 위치 | Instagram 피드 + Stories (자동 배치도 가능) |
| 타겟 연령 | 25~50세 |
| 타겟 성별 | 전체 |
| 타겟 관심사 | 인테리어, 홈데코, 부동산, 이사, 리모델링, 아파트 |
| 타겟 지역 | 대한민국 전체 (또는 서비스 가능 지역) |
| CTA 버튼 | "자세히 알아보기" |
| 랜딩 URL | 랜딩페이지 메인 URL + UTM 파라미터 |

### UTM 파라미터 예시

```
세트 A: ?utm_source=meta&utm_medium=instagram&utm_campaign=interior_test&utm_content=carousel_ba
세트 B: ?utm_source=meta&utm_medium=instagram&utm_campaign=interior_test&utm_content=single_emotion
```

## 6. 제작 체크리스트

### ChatGPT 이미지 생성 (3장)

- [ ] 카루셀 1장: 낡은 거실 Before 이미지
- [ ] 카루셀 2장: 완성된 거실 After 이미지
- [ ] 세트 B: 아름다운 완성 인테리어 이미지

### Canva 조립 (5장)

- [ ] 카루셀 1장: Before 이미지 + 텍스트 오버레이
- [ ] 카루셀 2장: After 이미지 + 텍스트 오버레이
- [ ] 카루셀 3장: 크림 배경 + Before/After 썸네일 + 체크리스트 (Canva 직접 제작)
- [ ] 카루셀 4장: 오렌지 그라데이션 + CTA (Canva 직접 제작)
- [ ] 세트 B: 완성 인테리어 + 텍스트 오버레이

### 최종 확인

- [ ] 모든 이미지 1080x1080px
- [ ] 텍스트 비율 20% 이하
- [ ] 오타 없음
- [ ] CTA 문구 명확함
- [ ] 랜딩 URL + UTM 정상 작동

## 7. 2주 후 성과 판단 기준

| 지표 | 좋음 | 보통 | 나쁨 (소재 교체) |
|------|------|------|-----------------|
| **CTR (클릭률)** | 2% 이상 | 1~2% | 1% 미만 |
| **CPC (클릭당 비용)** | 300원 이하 | 300~600원 | 600원 이상 |
| **CPL (리드당 비용)** | 5,000원 이하 | 5,000~10,000원 | 10,000원 이상 |
| **전환율** | 5% 이상 | 2~5% | 2% 미만 |

### 성과별 액션

```
CTR 높은데 전환 낮음 → 랜딩페이지 문제 (소재는 OK)
CTR 낮음 → 소재 교체 (이미지 또는 카피 변경)
둘 다 높음 → 예산 증액 (일 예산 2배로)
둘 다 낮음 → 타겟 변경 + 소재 전면 교체
```
