# 스토리 광고 레이아웃 가이드 — 유형별 여백 + 이미지 + 멘트 설계

> 작성일: 2026-04-08
> 상황: 인테리어 사진(가로형)을 9:16 캔버스에 넣으면 상하 여백이 생김
> 이 여백을 "광고 무기"로 활용하는 방법

---

## 기본 구조 이해

인테리어 사진(가로 또는 4:5)을 9:16 캔버스(1080x1920)에 넣으면 이렇게 된다:

```
┌──────────────────────┐
│                      │
│  ■ 상단 여백          │  약 350~450px
│    (훅 텍스트 영역)    │  → 1초 안에 시선을 잡는 한 마디
│                      │
├──────────────────────┤
│                      │
│  ■ 인테리어 사진       │  약 900~1100px
│    (메인 비주얼)       │  → 감정을 만드는 이미지
│                      │  → 이미지 위 오버레이는 최소한으로
│                      │
├──────────────────────┤
│                      │
│  ■ 하단 여백          │  약 350~450px
│    (설득 + CTA 영역)  │  → 행동을 끌어내는 마무리
│                      │
└──────────────────────┘
```

### 이 구조가 오히려 좋은 이유

- 사진만 꽉 채우면 텍스트가 이미지와 싸워서 가독성이 떨어짐
- **여백 = 텍스트 전용 공간** → 글씨가 깔끔하게 읽힘
- 상단에서 훅으로 멈추게 하고 → 사진으로 감정을 만들고 → 하단에서 행동시키는 **3단 설득 구조**가 자연스럽게 완성됨

## 여백 배경색 — 유형별 추천

배경색은 광고의 **첫인상 톤**을 결정한다. 유형별로 어울리는 색이 다르다.

| 유형 | 배경색 | HEX | 왜 이 색인가 |
|------|--------|-----|------------|
| 궁금증 유발 | **순수 블랙** | `#0A0A0A` | 어두울수록 비밀스러운 느낌. 흰 텍스트가 선명하게 떠오름 |
| 가격 충격 | **다크 네이비** | `#0D1B2A` | 고급스러움 + 신뢰감. 금액 숫자가 밝게 대비됨 |
| 나란히 비교 | **다크 차콜** | `#1A1A2E` | 중립적이고 깔끔. 비교 박스가 돋보이는 배경 |
| 손실 공포 | **딥 다크 레드** | `#1A0A0A` | 미세하게 붉은 기운 → 불안감을 은근히 조성 |
| 현실 공감 | **웜 크림** | `#FFF9F5` | 밝고 부드러움. 친구 SNS 느낌. 광고 같지 않음 |
| 사회적 증거 | **소프트 웜그레이** | `#F5F0EB` | 후기/리뷰 느낌. 깔끔하고 읽기 편함 |

> **팁**: 어두운 배경일수록 사진이 "빛나는" 느낌이 난다. 밝은 배경은 "일상적인" 느낌.

## 유형 1: 궁금증 유발형

### 감정 흐름: "뭐지?" → (사진 봄) → "어떻게?" → 클릭

### 배경: 순수 블랙 `#0A0A0A`

```
┌──────────────────────────┐
│  배경: #0A0A0A (블랙)      │
│                          │
│     인테리어 업체가         │  ← 흰색, 42px Bold
│     절대 먼저              │
│     말 안 하는 것          │
│                          │
├──────────────────────────┤
│                          │
│   ┌──────────────────┐   │
│   │                  │   │
│   │  (고급 인테리어    │   │  ← 사진 위에는 텍스트 없음
│   │   사진)           │   │     사진이 "답"처럼 보이게
│   │                  │   │
│   └──────────────────┘   │
│                          │
├──────────────────────────┤
│  배경: #0A0A0A (블랙)      │
│                          │
│   이 안에 답이 있습니다     │  ← 흰색 60%, 18px
│                          │
│   확인하기 →              │  ← 오렌지, 22px Bold
│                          │
└──────────────────────────┘
```

### 상단 여백 (훅)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 단색 블랙 | `#0A0A0A` |
| 훅 카피 | `인테리어 업체가`<br>`절대 먼저`<br>`말 안 하는 것` | 흰색 `#FFFFFF`, Pretendard Bold 42px, 가운데 정렬 |

> **핵심**: 상단 텍스트가 **질문을 던지지만 답을 안 준다**. 사진이 힌트처럼 보이고, 진짜 답은 랜딩페이지에.

### 이미지 위 오버레이

**없음.** 이 유형은 사진 위에 글씨를 넣지 않는 게 좋다.
사진 자체가 "이게 그 비밀의 결과물"처럼 느껴지게 하려면, 사진을 깨끗하게 보여줘야 한다.

### 하단 여백 (행동 유도)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 단색 블랙 | `#0A0A0A` (상단과 동일) |
| 서브 카피 | `이 안에 답이 있습니다` | 흰색 60% 투명도, 18px Medium |
| CTA | `확인하기 →` | 오렌지 `#FF6B00`, 22px Bold |

### 멘트 변형 (테스트용)

| 버전 | 상단 훅 | 하단 서브 |
|------|--------|----------|
| **v1** | `인테리어 업체가 절대 먼저 말 안 하는 것` | `이 안에 답이 있습니다` |
| v2 | `30평 인테리어를 1,600만원에 끝낸 비결` | `방법이 궁금하다면` |
| v3 | `이걸 모르면 인테리어 견적에서 당합니다` | `3분이면 확인됩니다` |

## 유형 2: 가격 충격형

### 감정 흐름: "예쁘다" → (가격 봄) → "진짜?! 이 가격에?" → 클릭

### 배경: 다크 네이비 `#0D1B2A`

```
┌──────────────────────────┐
│  배경: #0D1B2A (네이비)    │
│                          │
│     실제 시공 사례         │  ← 흰색 50%, 14px (태그)
│                          │
│     이 인테리어            │  ← 흰색, 40px Bold
│     2천만원 안에           │  ← "2천만원" 오렌지 46px
│     했다고?               │  ← 흰색, 40px Bold
│                          │
├──────────────────────────┤
│                          │
│   ┌──────────────────┐   │
│   │                  │   │
│   │  (고급스러운       │   │  ← 사진 하단에 살짝 그라데이션
│   │   인테리어 사진)   │   │
│   │                  │   │
│   └──────────────────┘   │
│                          │
├──────────────────────────┤
│  배경: #0D1B2A (네이비)    │
│                          │
│   비교견적 없었으면         │  ← 흰색 60%, 18px
│   4천만원 냈을 겁니다      │  ← "4천만원" 오렌지
│                          │
│   내 견적은 얼마일까? →    │  ← 오렌지, 22px Bold
│                          │
└──────────────────────────┘
```

### 상단 여백 (훅)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 다크 네이비 | `#0D1B2A` |
| 태그 | `실제 시공 사례` | 흰색 50%, 14px, 반투명 흰색 배경(15% 투명도) 태그 |
| 훅 1줄 | `이 인테리어` | 흰색, Bold 40px |
| 훅 2줄 | `2천만원 안에 했다고?` | 흰색 Bold 40px, `2천만원` → 오렌지 `#FF6B00` ExtraBold 46px |

> **핵심**: 금액이 **제일 먼저 눈에 들어와야** 한다. 사이즈와 색상 모두 가장 강하게.

### 이미지 위 오버레이

사진 하단에 **아주 얇은 그라데이션**(네이비→투명)만 살짝.
사진과 하단 여백이 자연스럽게 이어지도록 하는 용도. 텍스트는 없음.

### 하단 여백 (설득 + 행동)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 다크 네이비 | `#0D1B2A` |
| 서브 1줄 | `비교견적 없었으면` | 흰색 60%, 18px |
| 서브 2줄 | `4천만원 냈을 겁니다` | 흰색 60%, 18px, `4천만원` → 오렌지 |
| CTA | `내 견적은 얼마일까? →` | 오렌지, 22px Bold |

### 멘트 변형

| 버전 | 상단 훅 | 하단 서브 |
|------|--------|----------|
| **v1** | `이 인테리어 2천만원 안에 했다고?` | `비교견적 없었으면 4천만원 냈을 겁니다` |
| v2 | `이게 1,750만원이라고?` | `같은 스타일, 다른 업체는 3,100만원이었습니다` |
| v3 | `25평 인테리어 실제 견적` | `비교한 사람: 1,800만원 / 안 한 사람: 3,200만원` |

## 유형 3: 나란히 비교형 (절약 직격탄)

### 감정 흐름: "둘 다 같은 집?" → (숫자 비교) → "나는 어느 쪽이지?" → 클릭

### 배경: 다크 차콜 `#1A1A2E`

```
┌──────────────────────────┐
│  배경: #1A1A2E (차콜)      │
│                          │
│     같은 30평             │  ← 흰색, 36px Bold
│     같은 스타일            │  ← 흰색, 36px Bold
│                          │
│  ┌────────┐ ┌────────┐  │
│  │ A업체   │ │ B업체   │  │  ← 비교 박스 (여백 안에)
│  │3,600만원│ │1,980만원│  │
│  └────────┘ └────────┘  │
│                          │
├──────────────────────────┤
│                          │
│   ┌──────────────────┐   │
│   │                  │   │  ← 예쁜 인테리어 사진
│   │  (완성 인테리어)   │   │     "이게 둘 다 이 결과물"
│   │                  │   │
│   └──────────────────┘   │
│                          │
├──────────────────────────┤
│  배경: #1A1A2E (차콜)      │
│                          │
│     1,620만원             │  ← 오렌지, 52px ExtraBold
│     어디서 사라진 걸까요?   │  ← 흰색 70%, 22px
│                          │
│     비교하면 답이 보입니다 → │  ← 오렌지, 22px Bold
│                          │
└──────────────────────────┘
```

### 상단 여백 (훅 + 비교 박스)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 다크 차콜 | `#1A1A2E` |
| 훅 1줄 | `같은 30평` | 흰색, Bold 36px |
| 훅 2줄 | `같은 스타일` | 흰색, Bold 36px |
| A업체 박스 | `A업체` + `3,600만원` | 흰 배경, 둥근 모서리 12px, 라벨 16px 그레이, 금액 28px **빨간 취소선** |
| B업체 박스 | `B업체` + `1,980만원` | 오렌지 배경 `#FF6B00`, 라벨 16px 흰색, 금액 28px 흰색 Bold |
| 박스 크기 | 각 400 x 130px | 나란히 배치, 간격 40px |

> **핵심**: 비교 박스를 **사진 위가 아니라 상단 여백에** 넣는다. 사진은 깨끗하게 보여주고, 숫자 비교는 별도 공간에서 명확하게.

### 이미지 위 오버레이

**없음.** 사진을 깨끗하게 보여줘서 "이 결과물이 A든 B든 똑같다"는 걸 시각적으로 전달.

### 하단 여백 (임팩트 + 행동)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 다크 차콜 | `#1A1A2E` |
| 금액 (대형) | `1,620만원` | 오렌지 `#FF6B00`, ExtraBold 52px |
| 서브 | `어디서 사라진 걸까요?` | 흰색 70%, 22px |
| CTA | `비교하면 답이 보입니다 →` | 오렌지, 22px Bold |

### 멘트 변형

| 버전 | 상단 훅 | 하단 임팩트 |
|------|--------|-----------|
| **v1** | `같은 30평, 같은 스타일` | `1,620만원 어디서 사라진 걸까요?` |
| v2 | `같은 25평 아파트입니다` | `차이: 1,620만원. 비교 한 번이면 끝입니다` |
| v3 | `결과물은 똑같습니다` | `그런데 한 쪽은 1,620만원을 더 냈습니다` |

## 유형 4: 손실 공포형

### 감정 흐름: "무섭다" → (공사 현장 봄) → "나도 저렇게 될 수 있다" → 클릭

### 배경: 딥 다크 레드 `#1A0A0A`

```
┌──────────────────────────┐
│  배경: #1A0A0A (딥다크레드) │
│                          │
│     비교 없이              │  ← 흰색, 38px Bold
│     계약했다가             │
│                          │
│     1,200만원             │  ← 오렌지, 60px ExtraBold
│                          │
│     더 냈습니다            │  ← 흰색, 38px Bold
│                          │
├──────────────────────────┤
│                          │
│   ┌──────────────────┐   │
│   │                  │   │  ← 공사 현장 사진
│   │  (어두운 공사 현장  │   │     또는 하자 사진
│   │   또는 낡은 집)    │   │
│   │                  │   │
│   └──────────────────┘   │
│                          │
├──────────────────────────┤
│  배경: #1A0A0A (딥다크레드) │
│                          │
│   견적 비교 한 번이면       │  ← 흰색 60%, 18px
│   막을 수 있었습니다        │
│                          │
│   무료로 비교하기 →        │  ← 오렌지, 22px Bold
│                          │
└──────────────────────────┘
```

### 상단 여백 (공포 조성)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 딥 다크 레드 | `#1A0A0A` |
| 훅 1줄 | `비교 없이` | 흰색, Bold 38px |
| 훅 2줄 | `계약했다가` | 흰색, Bold 38px |
| 금액 (초대형) | `1,200만원` | 오렌지 `#FF6B00`, ExtraBold 60px |
| 훅 3줄 | `더 냈습니다` | 흰색, Bold 38px |

> **핵심**: 문장을 **3단으로 쪼개서** 금액이 중간에 홀로 서있게 만든다.
> "비교 없이 계약했다가" → (눈이 금액으로) → "1,200만원" → "더 냈습니다"
> 이 리듬이 공포감을 극대화한다.

### 이미지 위 오버레이

**없음.** 공사 현장 사진이 충분히 불안한 감정을 만들어준다. 텍스트를 얹으면 오히려 산만해짐.

### 하단 여백 (해결책 제시)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 딥 다크 레드 | `#1A0A0A` |
| 서브 1줄 | `견적 비교 한 번이면` | 흰색 60%, 18px |
| 서브 2줄 | `막을 수 있었습니다` | 흰색 60%, 18px |
| CTA | `무료로 비교하기 →` | 오렌지, 22px Bold |

> 상단이 공포라면, 하단은 **안도감**. "아직 늦지 않았다"는 느낌을 줘야 클릭한다.

### 멘트 변형

| 버전 | 상단 훅 | 하단 서브 |
|------|--------|----------|
| **v1** | `비교 없이 계약했다가 / 1,200만원 / 더 냈습니다` | `견적 비교 한 번이면 막을 수 있었습니다` |
| v2 | `인테리어 그냥 맡겼습니다 / 1,200만원 / 날렸습니다` | `이 실수, 3분이면 막을 수 있습니다` |
| v3 | `한 군데만 보고 계약한 결과 / 1,450만원 / 더 지불` | `비교 한 번이면 이런 일은 없었습니다` |

## 유형 5: 현실 공감형

### 감정 흐름: "이거 내 얘기네" → "나만 이런 게 아니었어" → "해결 방법 있나?" → 클릭

### 배경: 웜 크림 `#FFF9F5` (전체 단색, 사진 없음)

```
┌──────────────────────────┐
│  배경: #FFF9F5 (크림) 전체  │
│                          │
│  (상단 세이프존)            │
│                          │
│                          │
│     인테리어 견적           │  ← 다크그레이, 36px Medium
│     3군데 받았는데          │
│                          │
│     다 다르게 나와서        │
│     진짜 멘붕              │  ← "멘붕" 오렌지, 42px Bold
│                          │
│     어디가 정상인지         │
│     어떻게 알아요...        │
│                          │
│                          │
│     ─────────────         │  ← 얇은 구분선 (그레이 20%)
│                          │
│     이 고민 공감되면        │  ← 오렌지, 24px Bold
│     눌러보세요 ↑           │
│                          │
│  (하단 세이프존)            │
│                          │
└──────────────────────────┘
```

### 이 유형은 상단/이미지/하단 구분이 없다

사진을 쓰지 않기 때문에 **전체가 하나의 텍스트 카드**다.

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 전체 크림 단색 | `#FFF9F5` |
| 텍스트 1~2줄 | `인테리어 견적` / `3군데 받았는데` | 다크그레이 `#1A1A1A`, Medium 36px |
| 텍스트 3~4줄 | `다 다르게 나와서` / `진짜 멘붕` | 다크그레이, Medium 36px / `멘붕`만 오렌지 `#FF6B00` Bold 42px |
| 텍스트 5~6줄 | `어디가 정상인지` / `어떻게 알아요...` | 다크그레이, Medium 36px |
| 구분선 | 가로 200px 직선 | 그레이 `#DDDDDD`, 중앙 |
| CTA | `이 고민 공감되면` / `눌러보세요 ↑` | 오렌지 `#FF6B00`, Bold 24px |

> **핵심**: 이 유형은 **광고 같지 않은 게 무기**. 예쁜 사진도 없고, 브랜드 로고도 없고, 그냥 누군가 답답해서 올린 글처럼 보여야 한다.

### 멘트 변형

| 버전 | 메인 텍스트 | CTA |
|------|-----------|-----|
| **v1** | `견적 3군데 다 다르게... 진짜 멘붕 / 어디가 정상인지 어떻게 알아요...` | `이 고민 공감되면 눌러보세요` |
| v2 | `인테리어 업체 고르는 거 왜 이렇게 어렵죠 / 후기 봐도 다 광고 같고 / 견적은 천차만별이고...` | `같은 고민이면 여기 보세요` |
| v3 | `솔직히 인테리어 처음이라 뭐부터 해야 될지 모르겠음 / 견적은 얼마가 맞는 건지 / 업체는 어떻게 골라야 하는 건지` | `정리해드릴게요` |

## 유형 6: 사회적 증거형 (후기)

### 감정 흐름: "이 사람도 나랑 같은 고민이었네" → "800만원 아꼈다고?" → "나도 해볼까" → 클릭

### 배경: 소프트 웜그레이 `#F5F0EB`

```
┌──────────────────────────┐
│  배경: #F5F0EB (웜그레이)   │
│                          │
│  (상단 세이프존)            │
│                          │
│  ┌──────────────────┐    │
│  │  💬               │    │
│  │ "솔직히 귀찮아서    │    │  ← 말풍선 카드
│  │  한 군데만         │    │     흰 배경, 둥근 모서리, 그림자
│  │  맡기려다가        │    │
│  │  여기서 비교했는데  │    │
│  │                  │    │
│  │  800만원 아꼈어요  │    │  ← 오렌지, Bold
│  │  진짜로"          │    │
│  │                  │    │
│  │  — 서울 마포구     │    │  ← 그레이, 작게
│  │    32평 리모델링   │    │
│  └──────────────────┘    │

├──────────────────────────┤

│   ┌──────────────────┐   │
│   │                  │   │  ← 완성된 인테리어 사진
│   │  (예쁜 완성 거실)  │   │     "이게 그 사람의 결과물"
│   │                  │   │
│   └──────────────────┘   │
│                          │
├──────────────────────────┤
│  배경: #F5F0EB (웜그레이)   │
│                          │
```

### 상단 여백 (후기 카드)

| 요소 | 내용 | 스타일 |
|------|------|--------|
| 배경 | 소프트 웜그레이 | `#F5F0EB` |
| 말풍선 카드 | 흰색 사각형 | 배경 `#FFFFFF`, 둥근 모서리 20px, 그림자(검정 8%, Y:4px, 블러:12px) |
| 카드 크기 | 900 x 520px | 중앙 배치 |
| 후기 본문 | `솔직히 귀찮아서 한 군데만 맡기려다가 여기서 비교했는데` | 다크그레이 `#1A1A1A`, Medium 22px |
| 후기 금액 | `800만원 아꼈어요 진짜로` | 오렌지 `#FF6B00`, Bold 32px |
| 후기 출처 | `— 서울 마포구, 32평 리모델링` | 그레이 `#888888`, 14px |

### 이미지 위 오버레이

**없음.** 예쁜 인테리어 사진을 깨끗하게 보여줘서 "후기 쓴 사람의 실제 결과물"처럼 느끼게 한다.

## 전체 요약 — 한눈에 비교

| 유형 | 배경색 | 상단 역할 | 이미지 위 | 하단 역할 |
|------|--------|----------|----------|----------|
| 궁금증 | 블랙 `#0A0A0A` | 질문 (답 안 줌) | 텍스트 없음 | 힌트 + CTA |
| 가격 충격 | 네이비 `#0D1B2A` | 가격 숫자 충격 | 텍스트 없음 | 비교 금액 + CTA |
| 나란히 비교 | 차콜 `#1A1A2E` | 조건 동일 + 비교 박스 | 텍스트 없음 | 차액 숫자 + CTA |
| 손실 공포 | 딥다크레드 `#1A0A0A` | 3단 분리 (훅-금액-결과) | 텍스트 없음 | 해결책 + CTA |
| 현실 공감 | 크림 `#FFF9F5` | (사진 없음, 전체 텍스트) | — | 공감 + CTA |
| 사회적 증거 | 웜그레이 `#F5F0EB` | 말풍선 후기 카드 | 텍스트 없음 | 행동 유도 + CTA |

### 공통 패턴

1. **이미지 위에는 텍스트를 넣지 않는다** — 인테리어 사진은 깨끗하게 보여줘야 퀄리티가 살아남
2. **상단 여백 = 시선을 잡는 훅** — 1초 안에 읽히는 짧은 문장
3. **하단 여백 = 행동을 끌어내는 마무리** — 서브 한 줄 + CTA
4. **금액 숫자는 항상 오렌지 + 가장 큰 사이즈** — 숫자가 제일 먼저 눈에 들어와야 함
5. **사진과 여백 경계는 그라데이션으로 부드럽게** — 사진 상하단에 배경색→투명 얇은 그라데이션

### 그라데이션 경계 처리 (Canva)

사진과 여백이 딱 끊기면 어색하다. 사진 위아래에 얇은 그라데이션을 깔면 자연스러워진다.

1. 사각형 추가 (1080 x 100px)
2. 색상: 상단/하단 여백과 동일한 색
3. 투명도 그라데이션 적용 (배경색 100% → 0%)
4. 사진 위/아래 경계에 겹쳐서 배치
