## F1. 궁금증 유발 — 비결 공개형

### Step 1: AI 이미지 생성

기존 A1 이미지가 있으므로 **Canva에서 리사이즈**하거나, 아래 프롬프트로 새로 생성.

```
고급스러운 한국 아파트 30평 거실+주방이 모던하게 완성된 인테리어 사진.

- 화이트 벽면, 밝은 원목 바닥
- 아일랜드 식탁 위 황동 팬던트 조명
- 라이트 베이지 소파, 큰 창으로 따뜻한 자연광
- 그린 식물 포인트
- 인테리어 매거진 스타일, 와이드 정면 구도
- 세로 4:5 비율 (1080x1350px), 고해상도
- 하단 35%는 텍스트용 어두운 그라데이션
```

### Step 2: Canva 텍스트 작업

캔버스 크기: **1080 x 1350px**

| 요소 | 내용 | 폰트/크기 | 색상 | 위치 |
|------|------|----------|------|------|
| 메인 카피 1줄 | `30평 인테리어를` | Pretendard Bold 38px | 흰색 #FFFFFF | 하단에서 위로 280px |
| 메인 카피 2줄 | `1,600만원에 끝낸 방법` | Pretendard Bold 38px | 흰색 #FFFFFF | 하단에서 위로 220px |
| 금액 강조 | `1,600만원` 부분만 | Pretendard ExtraBold 42px | 오렌지 #FF6B00 | 위 텍스트 안에서 색만 변경 |
| 배경 처리 | 하단 그라데이션 | — | 검정→투명 | 하단 35% 영역 |

**Canva 작업 순서:**
1. 1080x1350 캔버스 생성
2. AI 생성 이미지를 배경으로 넣기 (꽉 채우기)
3. 하단에 검정→투명 그라데이션 사각형 추가 (투명도 70%)
4. 텍스트 입력, 가운데 정렬
5. `1,600만원` 부분만 오렌지로 색상 변경
6. PNG로 다운로드


## 폰트 (Canva 검색어)

| 용도 | 폰트 | Canva 검색어 |
|------|------|-------------|
| 헤드라인 (큰 글씨) | Pretendard Bold | `프리텐다드` 또는 `Noto Sans KR Bold` |
| 본문/서브 | Pretendard Medium | `프리텐다드` 또는 `Noto Sans KR Medium` |
| 숫자 강조 | Pretendard ExtraBold | 같은 폰트 굵기만 ExtraBold로 |

> Canva에서 Pretendard가 안 보이면 `Noto Sans KR`로 대체. 거의 동일함.

## 그라데이션 오버레이 만드는 법 (Canva)

1. 사각형 추가 → 이미지 하단에 배치
2. 사각형 색상: 검정 #000000
3. 사각형 높이: 캔버스 높이의 35~50%
4. **투명도**: "그라데이션" 효과 적용
   - Canva에서는 사각형 선택 → 투명도 70%로 설정
   - 또는 "그래디언트" 소재를 검색해서 사용
5. 위치: 캔버스 하단에 정렬

## 텍스트 그림자 설정 (사진 위 흰 텍스트용)

1. 텍스트 선택
2. 효과(Effects) → 그림자(Shadow)
3. 오프셋: 2px
4. 블러: 8px
5. 색상: 검정
6. 투명도: 50%

## 최종 다운로드 설정

- 파일 형식: **PNG** (텍스트 선명도 유지)
- 크기: 자동 (캔버스 크기 그대로)
- 투명 배경: OFF
