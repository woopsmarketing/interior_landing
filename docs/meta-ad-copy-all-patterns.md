# 인스타그램 광고 소재 전체 패턴 — 고CTR 최적화 버전

> 목적: 모든 패턴을 순서대로 테스트 → CTR/전환율 상위 2~3개만 살려서 예산 집중
> 운영 방식: 패턴당 일 4,000~5,000원 × 3~4일 → 성과 비교 → 승자에 예산 몰빵

---

## 패턴 A — "궁금증 유발형" ★★★★★ (1순위 추천)

> 답을 주지 않는다. 클릭해야만 알 수 있게 만든다.
> 텍스트가 짧을수록 강력하다.

---

### A-1 "비결 공개형"

**이미지**
```
고급스러운 완성 인테리어 사진 (밝고 넓어 보이는 거실+주방)
하단에 어두운 그라데이션
```

**텍스트 오버레이**
```
큰 글씨:
"30평 인테리어를
1,600만원에 끝낸 방법"

(그 이상 설명 없음)
```

**왜 클릭하는가**
- "어떻게?" 라는 질문이 자동으로 생김
- 클릭 안 하면 손해 볼 것 같은 느낌
- 광고처럼 안 보이고 정보처럼 보임

---

### A-2 "숨겨진 정보형"

**이미지**
```
예쁜 인테리어 완성 사진
```

**텍스트 오버레이**
```
큰 글씨:
"인테리어 업체들이
절대 먼저 알려주지 않는 것"
```

**왜 클릭하는가**
- "나만 모르는 게 있다" 심리 자극
- "업체가 숨긴다" → 배신감 + 궁금증 동시 유발

---

### A-3 "수치 미완성형"

**이미지**
```
리모델링 완성된 밝고 깔끔한 거실 사진
```

**텍스트 오버레이**
```
큰 글씨:
"같은 25평, 견적 차이가
____만원이었습니다"

작은 글씨:
"얼마인지 확인하기 →"
```

**왜 클릭하는가**
- 빈칸은 채우고 싶은 본능을 자극
- 숫자가 궁금해서 클릭할 수밖에 없음

---

## 패턴 B — "가격 충격형" ★★★★★ (1순위 추천)

> 한국에서 가장 강력한 트리거는 돈이다.
> 예상보다 싸거나, 예상보다 많이 차이 나거나.

---

### B-1 "이게 얼마라고?"

**이미지**
```
누가 봐도 고급스러운 인테리어 사진
(대리석 느낌 주방, 간접조명, 넓어 보이는 거실)
```

**텍스트 오버레이**
```
상단 작게:
"실제 시공 사례"

큰 글씨:
"이 인테리어
1,750만원에 했다고?"

하단 작게:
"비교 견적 없었으면 3,100만원 냈을 겁니다"
```

**왜 클릭하는가**
- "진짜?" 반응 → 즉각 클릭
- "나도 저렇게 할 수 있겠다" 희망 자극

---

### B-2 "나란히 비교"

**이미지**
```
완성 인테리어 사진 (고급스러운 느낌)
좌우로 텍스트 분리 배치
```

**텍스트 오버레이**
```
큰 글씨 (좌측):         큰 글씨 (우측):
"A업체"                 "B업체"
"3,600만원"             "1,980만원"

중앙 하단 크게:
"같은 25평, 같은 스타일"

아주 작게:
"어디서 1,620만원이 사라진 걸까요?"
```

**왜 클릭하는가**
- 숫자 비교는 뇌가 자동으로 반응
- "나는 어느 쪽에 해당될까?" → 클릭

---

### B-3 "절약 금액 직격탄"

**이미지**
```
밝고 완성된 인테리어 배경
하단 그라데이션
```

**텍스트 오버레이**
```
큰 글씨:
"견적 비교 전: 3,400만원
견적 비교 후: 1,950만원"

강조 텍스트 (오렌지):
"절약한 금액: 1,450만원"

하단:
"3분 폼 하나 차이입니다"
```

**왜 클릭하는가**
- 1,450만원이라는 구체적인 숫자 → 현실감
- "나도 저만큼 낭비하고 있나?" 불안 유발

---

## 패턴 C — "손실 공포형" ★★★★☆ (2순위)

> 사람은 얻는 것보다 잃는 것에 2배 더 민감하게 반응한다.
> "당신은 지금 손해 보고 있다"는 메시지.

---

### C-1 "이미 당한 사람들"

**이미지**
```
예쁜 인테리어 완성 사진 (배경)
무거운 분위기의 텍스트
```

**텍스트 오버레이**
```
큰 글씨:
"인테리어 그냥 맡겼다가
1,200만원 더 냈습니다"

작은 글씨:
"견적 비교 한 번이면
막을 수 있었던 일입니다"

하단 오렌지:
"지금 무료로 비교하기 →"
```

**왜 클릭하는가**
- "나도 저렇게 될 수 있다" 두려움
- 해결책이 바로 아래에 있어서 클릭 동기 명확

---

### C-2 "지금 안 하면 손해"

**이미지**
```
인테리어 공사 현장 사진 (반쯤 진행된 상태)
또는 낡은 집 사진
```

**텍스트 오버레이**
```
큰 글씨:
"인테리어 업체 고를 때
비교 안 한 사람 손들어보세요"

작은 글씨:
"평균 600~1,500만원을
그냥 더 냈을 가능성이 높습니다"
```

---

### C-3 "경고형"

**이미지**
```
크림색 배경 텍스트 카드 (사진 없이 텍스트만)
```

**텍스트**
```
큰 글씨:
"인테리어 계약 전에
이것만큼은 꼭 확인하세요"

리스트:
□ 견적 2곳 이상 비교했는가
□ AI로 완성 모습을 먼저 봤는가
□ 원하는 업체를 직접 선택했는가

하단:
"3가지 다 무료로 해드립니다 →"
```

**왜 클릭하는가**
- 체크리스트는 자신을 점검하게 만듦
- "나는 아직 안 했는데" → 불안 → 클릭

---

## 패턴 D — "현실 공감형" ★★★★☆ (2순위)

> 광고처럼 안 보이는 게 최고의 광고다.
> 인스타 피드에 자연스럽게 녹아드는 텍스트 카드.

---

### D-1 "내 얘기 같은 카피"

**이미지**
```
심플한 크림색(#FFF9F5) 배경
텍스트만 있는 카드
마치 누군가의 SNS 게시물처럼
```

**텍스트**
```
"인테리어 견적 3군데 받았는데
다 다르게 나와서 진짜 멘붕옴

어디가 정상인지 어떻게 알아요...

→ 이 고민 공감되면 눌러보세요"
```

**왜 클릭하는가**
- 광고 같지 않음 → 경계심 없음
- "내 얘기네" → 공감 → 클릭
- 댓글, 공유도 많이 발생하는 포맷

---

### D-2 "질문형"

**이미지**
```
예쁜 인테리어 배경 + 심플한 텍스트
```

**텍스트 오버레이**
```
큰 글씨:
"혹시 인테리어 견적
한 군데만 받으셨나요?"

작은 글씨:
"그게 비싼 건지 싼 건지
비교 없이는 알 수가 없습니다"
```

---

### D-3 "이사/신혼 타겟"

**이미지**
```
밝고 깔끔한 신혼집 느낌 인테리어
```

**텍스트 오버레이**
```
큰 글씨:
"이사 앞두고 계신가요?"

작은 글씨:
"인테리어 업체 고르는 거
생각보다 훨씬 복잡합니다

저희가 대신 정리해드릴게요"
```

## 패턴 E — "AI 호기심 자극형" ★★★★☆ (차별화 포인트)

> 우리 플랫폼만의 고유 기능.
> "AI가 미리 보여준다"는 건 다른 업체에 없는 무기.

---

### E-1 "마법 같은 변신"

**이미지**
```
왼쪽: 낡은 거실 사진 (실제 사진 느낌)
오른쪽: 같은 공간이 AI로 변환된 완성 모습
가운데 화살표(→) 또는 세로 구분선
```

**텍스트 오버레이**
```
상단:
"사진 1장을 올리면"

하단 크게:
"AI가 완성된 모습을
미리 보여드립니다"

아주 작게:
"무료 · 3분이면 확인 가능"
```

**왜 클릭하는가**
- Before/After 비주얼 충격
- "내 집도 해보고 싶다" → 즉각 클릭 욕구

---

### E-2 "AI 기술 자극"

**이미지**
```
고급 인테리어 완성 이미지
"AI Generated" 워터마크 느낌으로 작게 표기
```

**텍스트 오버레이**
```
큰 글씨:
"공사 시작 전에
완성된 모습을 먼저 볼 수 있다면?"

작은 글씨:
"AI가 우리 집 인테리어를
미리 그려드립니다"

하단 오렌지:
"무료로 미리보기 →"
```

## 전체 테스트 일정 및 예산 배분

### 1주차: 전체 패턴 테스트 (일 예산 17,000원)

| 날짜 | 집행 소재 | 일 예산 |
|------|----------|---------|
| 1~3일 | A-1, B-1, C-1 (각 5,000원) | 15,000원 |
| 4~6일 | D-1, E-1, F-2 (각 5,000원) | 15,000원 |
| 7일 | 성과 분석 | — |

### 2주차: 상위 3개만 집중 (일 예산 17,000원)

```
CTR 상위 3개 소재 × 일 5,600원
```

### 성과 판단 기준 (3일 후 컷)

| 지표 | 살리기 | 교체 |
|------|--------|------|
| CTR | 1.5% 이상 | 1% 미만 |
| CPC | 400원 이하 | 700원 이상 |
| 전환율 | 3% 이상 | 1% 미만 |

## 광고 공통 설정

```
CTA 버튼: "자세히 알아보기"
랜딩 URL: 메인 랜딩페이지 + UTM 파라미터
노출 위치: 인스타그램 피드 + 스토리 (자동 배치)
타겟: 25~50세, 인테리어/부동산/이사/신혼 관심사
```
