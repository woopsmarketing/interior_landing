# Meta(Instagram) 광고 실행 플랜

> 작성일: 2026-04-07 (업데이트: 2026-04-08)
> 목적: 두 광고 문서를 어떻게 활용할지 정리 + 실행 순서 명확화

---

## Instagram 광고 배치별 이미지 규격

| 배치 | 설명 | 권장 비율 | 권장 해상도 |
|------|------|----------|-----------|
| **피드** | 메인 화면 스크롤 중 일반 게시물 사이에 노출 | **4:5** | **1080 x 1350px** |
| **스토리** | 스토리 사이에 끼어드는 세로 풀스크린 광고 | **9:16** | **1080 x 1920px** |
| **릴스** | 릴스 영상 사이에 삽입되는 광고 | **9:16** | **1080 x 1920px** |
| **카루셀** | 2~10장을 옆으로 넘기며 보는 형태 | **1:1** 또는 **4:5** | **1080x1080** 또는 **1080x1350** |

> 스토리/릴스는 상단·하단에 인스타 UI가 겹치므로 가장자리에 중요 텍스트를 넣지 않아야 함.
> "자동 배치" 선택 시 Meta가 피드/스토리/릴스에 알아서 노출.

## Phase 1: 소재 제작 (1~2일)

`creative-guide.md` 6장 "제작 체크리스트" 기준으로 진행.

### Step 1 — ChatGPT 이미지 생성 (3장)

| 이미지 | 용도 | 프롬프트 위치 |
|--------|------|-------------|
| 낡은 거실 Before | 카루셀 1장 배경 | creative-guide.md > 카루셀 1장 > 이미지 생성 프롬프트 |
| 완성 거실 After | 카루셀 2장 배경 | creative-guide.md > 카루셀 2장 > 이미지 생성 프롬프트 |
| 아름다운 완성 인테리어 | 세트 B 싱글 이미지 배경 | creative-guide.md > 세트 B > 이미지 생성 프롬프트 |

> 팁: ChatGPT에 프롬프트를 그대로 복붙하면 됨. 1:1 정사각형, 고해상도로 생성.

### Step 2 — Canva 조립 (5장)

| 소재 | 제작 방식 | 가이드 위치 |
|------|----------|------------|
| 카루셀 1장 (Before) | AI 이미지 + 텍스트 오버레이 | creative-guide.md > 카루셀 1장 > Canva 텍스트 가이드 |
| 카루셀 2장 (After) | AI 이미지 + 텍스트 오버레이 | creative-guide.md > 카루셀 2장 > Canva 텍스트 가이드 |
| 카루셀 3장 (플랫폼 소개) | Canva 직접 제작 (크림 배경) | creative-guide.md > 카루셀 3장 > Canva 제작 가이드 |
| 카루셀 4장 (CTA) | Canva 직접 제작 (오렌지 배경) | creative-guide.md > 카루셀 4장 > Canva 제작 가이드 |
| 세트 B 싱글 | AI 이미지 + 텍스트 오버레이 | creative-guide.md > 세트 B > Canva 텍스트 가이드 |

### Step 3 — 최종 확인

- [ ] 피드 이미지: **1080 x 1350px (4:5)** — 피드에서 가장 넓은 면적 차지
- [ ] 카루셀 이미지: 1080 x 1080px (1:1) 또는 1080 x 1350px (4:5) — 전체 카드 동일 비율
- [ ] 스토리 이미지: **1080 x 1920px (9:16)** — `instagram-story-ad-guide.md` 참조
- [ ] 텍스트 비율 20% 이하 (Meta 정책 — 초과 시 도달 감소)
- [ ] 오타 없음
- [ ] CTA 문구 명확함 ("자세히 알아보기")

## Phase 2: 캠페인 세팅 (30분)

`creative-guide.md` 5장 "광고 캠페인 설정 가이드" 기준.

### 캠페인 구조

```
캠페인: "인테리어 견적 비교 — 테스트"
├── 광고 세트 A: 카루셀 4장 (일 8,500원)
└── 광고 세트 B: 싱글 이미지 (일 8,500원)
```

### 세팅 값

| 항목 | 값 |
|------|-----|
| 캠페인 목표 | 전환 (웹사이트) 또는 트래픽 |
| 일 예산 | 8,500원 x 2세트 = 17,000원/일 |
| 노출 위치 | Instagram 피드 + Stories |
| 타겟 연령 | 25~50세 |
| 타겟 관심사 | 인테리어, 홈데코, 부동산, 이사, 리모델링 |
| 타겟 지역 | 대한민국 전체 |
| CTA 버튼 | "자세히 알아보기" |

### UTM 파라미터

```
세트 A: ?utm_source=meta&utm_medium=instagram&utm_campaign=interior_test&utm_content=carousel_ba
세트 B: ?utm_source=meta&utm_medium=instagram&utm_campaign=interior_test&utm_content=single_emotion
```

- [ ] 랜딩 URL + UTM 파라미터 클릭 시 정상 작동 확인

## Phase 3: 2주 A/B 테스트 운영

### 성과 판단 기준 (2주 후)

| 지표 | 좋음 | 보통 | 나쁨 (소재 교체) |
|------|------|------|-----------------|
| CTR (클릭률) | 2% 이상 | 1~2% | 1% 미만 |
| CPC (클릭당 비용) | 300원 이하 | 300~600원 | 600원 이상 |
| CPL (리드당 비용) | 5,000원 이하 | 5,000~10,000원 | 10,000원 이상 |
| 전환율 | 5% 이상 | 2~5% | 2% 미만 |

### 성과별 액션

| 상황 | 원인 | 액션 |
|------|------|------|
| CTR 높은데 전환 낮음 | 랜딩페이지 문제 | 랜딩페이지 개선 (소재는 유지) |
| CTR 낮음 | 소재가 안 먹힘 | → **Phase 4로 이동** (소재 교체) |
| 둘 다 높음 | 성공 | 예산 증액 (일 예산 2배) |
| 둘 다 낮음 | 타겟 + 소재 둘 다 문제 | 타겟 변경 + 소재 전면 교체 |

## Phase 4: 소재 교체 (성과 미달 시)

**이 단계에서 `meta-ad-copy-all-patterns.md`를 꺼낸다.**

### 교체 우선순위 TOP 3

| 순위 | 패턴 | 카피 핵심 | 왜 강력한가 |
|------|------|----------|------------|
| 1위 | A-1 궁금증 유발 | "30평 인테리어를 1,600만원에 끝낸 방법" | 궁금증 + 가격 + 정보성 위장 3박자 |
| 2위 | B-1 가격 충격 | "이 인테리어 1,750만원에 했다고?" | 가격 충격 + 고급 사진 조합 |
| 3위 | D-1 현실 공감 | "견적 3군데 받았는데 다 다르게 나와서 멘붕" | 광고 같지 않아서 경계심 제로 |

### 교체 테스트 방법 (all-patterns.md 기준)

```
패턴당 일 4,000~5,000원 x 3~4일 → 성과 비교 → 상위 2~3개에 예산 집중
```

| 날짜 | 테스트 소재 | 일 예산 |
|------|-----------|---------|
| 1~3일 | A-1, B-1, C-1 (각 5,000원) | 15,000원 |
| 4~6일 | D-1, E-1, F-2 (각 5,000원) | 15,000원 |
| 7일 | 성과 분석 → 상위 3개 선정 | — |
| 8일~ | 상위 3개만 집중 운영 | 17,000원 |

### 추가 교체 변형 (all-patterns.md 하위 변형)

Phase 4에서도 성과 미달 시, 같은 패턴의 다른 변형을 테스트:

- A 패턴: A-2 "업체들이 절대 먼저 알려주지 않는 것", A-3 "같은 25평, 견적 차이가 ____만원"
- B 패턴: B-2 "A업체 vs B업체 나란히 비교", B-3 "절약 금액 직격탄"
- E 패턴: E-1 "Before→After AI 변신" (우리 플랫폼만의 차별화 포인트)

## 현재 피드 이미지 리사이즈 필요

현재 A1~B2 이미지가 **1536x1024 (3:2 가로형)**으로 되어 있어 피드 권장 비율에 맞지 않음.

| 이미지 | 현재 크기 | 필요한 크기 | 리사이즈 방법 |
|--------|----------|-----------|-------------|
| A1, A2, B1 | 1536x1024 | **1080x1350 (4:5)** | Canva 크롭 가능 (텍스트 하단/중앙 배치) |
| A3 (Before/After 좌우) | 1536x1024 | **1080x1350 (4:5)** | 좌우 분할이라 **다시 생성 필요** |
| B2 (A업체/B업체 좌우) | 1536x1024 | **1080x1350 (4:5)** | 좌우 배치라 **다시 생성 필요** |

> 4:5 세로형이 피드에서 1:1보다 면적이 넓어 클릭률이 더 높음.
> ChatGPT/Gemini에서 다시 생성할 때 "세로 4:5 비율 (1080x1350px)" 명시.
