기본 구조 이해

인테리어 사진(가로 또는 4:5)을 9:16 캔버스(1080x1920)에 넣으면 이렇게 된다:

┌──────────────────────┐ │ │ │ ■ 상단 여백 │ 약 350~450px │ (훅 텍스트 영역) │ → 1초 안에 시선을 잡는 한 마디 │ │ ├──────────────────────┤ │ │ │ ■ 인테리어 사진 │ 약 900~1100px │ (메인 비주얼) │ → 감정을 만드는 이미지 │ │ → 이미지 위 오버레이는 최소한으로 │ │ ├──────────────────────┤ │ │ │ ■ 하단 여백 │ 약 350~450px │ (설득 + CTA 영역) │ → 행동을 끌어내는 마무리 │ │ └──────────────────────┘

이 구조가 오히려 좋은 이유

  • 사진만 꽉 채우면 텍스트가 이미지와 싸워서 가독성이 떨어짐
  • 여백 = 텍스트 전용 공간 → 글씨가 깔끔하게 읽힘
  • 상단에서 훅으로 멈추게 하고 → 사진으로 감정을 만들고 → 하단에서 행동시키는 3단 설득 구조가 자연스럽게 완성됨

여백 배경색 — 유형별 추천

배경색은 광고의 첫인상 톤을 결정한다. 유형별로 어울리는 색이 다르다.

유형배경색HEX왜 이 색인가
궁금증 유발 순수 블랙 #0A0A0A 어두울수록 비밀스러운 느낌. 흰 텍스트가 선명하게 떠오름
가격 충격 다크 네이비 #0D1B2A 고급스러움 + 신뢰감. 금액 숫자가 밝게 대비됨
나란히 비교 다크 차콜 #1A1A2E 중립적이고 깔끔. 비교 박스가 돋보이는 배경
손실 공포 딥 다크 레드 #1A0A0A 미세하게 붉은 기운 → 불안감을 은근히 조성
현실 공감 웜 크림 #FFF9F5 밝고 부드러움. 친구 SNS 느낌. 광고 같지 않음
사회적 증거 소프트 웜그레이 #F5F0EB 후기/리뷰 느낌. 깔끔하고 읽기 편함
팁: 어두운 배경일수록 사진이 "빛나는" 느낌이 난다. 밝은 배경은 "일상적인" 느낌.

공통 패턴

  1. 1 이미지 위에는 텍스트를 넣지 않는다 — 인테리어 사진은 깨끗하게 보여줘야 퀄리티가 살아남
  2. 2 상단 여백 = 시선을 잡는 훅 — 1초 안에 읽히는 짧은 문장
  3. 3 하단 여백 = 행동을 끌어내는 마무리 — 서브 한 줄 + CTA
  4. 4 금액 숫자는 항상 오렌지 + 가장 큰 사이즈 — 숫자가 제일 먼저 눈에 들어와야 함
  5. 5 사진과 여백 경계는 그라데이션으로 부드럽게 — 사진 상하단에 배경색→투명 얇은 그라데이션
