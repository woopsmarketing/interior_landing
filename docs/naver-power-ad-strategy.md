# 네이버 파워광고 전략 가이드
> 서비스: AI 인테리어 견적 비교 플랫폼
> 작성일: 2026-03-20
> 목표: 빠른 테스트 → 품질지수 개선 → 단계적 확장

---

## 1. 현재 상황 진단

| 항목 | 현재 | 문제 |
|------|------|------|
| 키워드 | 대구인테리어 | 지역 업체들이 몰리는 레드오션 |
| 입찰가 | 자동 ~100원 | 경쟁 대비 현저히 낮음 |
| 광고 순위 | 3페이지 하단 | 노출 자체가 거의 없음 |
| 랜딩 관련성 | 전국 일반 페이지 | 키워드와 관련성 낮아 품질지수 약함 |

### 핵심 원인
네이버 파워링크 순위 = **품질지수 × 입찰가**
품질지수는 크게 세 가지로 결정됨:
- 키워드 ↔ 광고 소재 관련성
- 광고 소재 ↔ 랜딩페이지 관련성
- 광고효과(CTR, 클릭률)

지금은 셋 다 약한 상태.

## 2. 포지셔닝 재정의

> "우리는 대구 인테리어 업체가 아니라, 인테리어 견적을 한 번에 비교해주는 플랫폼이다."

이 차이를 광고 전략에 반드시 반영해야 한다.
**지역 업체의 전략을 따라가면 안 된다.**

| 구분 | 지역 인테리어 업체 | 우리 서비스 |
|------|------------------|------------|
| 핵심 가치 | "저희 업체 선택해주세요" | "여러 업체를 한 번에 비교하세요" |
| 타겟 키워드 | 대구인테리어, 강남인테리어 | 인테리어견적비교, 인테리어업체비교 |
| 경쟁 강도 | 매우 높음 | 상대적으로 낮음 |
| 전환 의도 | 다양함 | 높음 (비교 의도 = 견적 의지 강함) |

## 3. 키워드 전략

### 그룹 A: 핵심 — 비교/견적 의도 키워드 ⭐ (최우선)

경쟁이 적고 전환율이 높은 블루오션. 예산의 60% 배분.

```
인테리어견적비교
인테리어업체비교
인테리어견적한번에
인테리어무료견적비교
아파트인테리어견적비교
인테리어견적받기
인테리어여러군데견적
```

**입찰가**: 수동 300~500원으로 시작

---

### 그룹 B: 지역+의도 조합 키워드 (보조)

"대구인테리어"보다 경쟁이 훨씬 적고, 비교 의도가 있는 사람을 정확히 타겟팅.

```
대구아파트인테리어견적
서울인테리어견적비교
부산인테리어견적받기
수원인테리어업체비교
```

**입찰가**: 수동 200~400원
**중요**: 지역명은 2~3개 주요 도시만 (아래 별도 설명)

---

### 그룹 C: 보류 — 레드오션 (지금 당장은 비효율)

```
대구인테리어     → 지역 업체들 입찰 극심, 품질 올라간 후 진입
인테리어         → 너무 광범위, 전환율 낮음
인테리어견적     → 경쟁 높음, 나중에 예산 확보 후
```

## 4. 광고 소재 전략

### AI 미리보기 차별점 활용 방법

> Q: 사람들이 "AI 미리보기"를 검색하진 않는데, 이걸 어떻게 써야 할까?
> A: **키워드 ≠ 소재**. 키워드는 인테리어 관련이지만, 소재(광고 문구)에서 AI 차별점을 보여준다.

**역할 분리:**
- **키워드** → 검색 의도에 맞추기 (인테리어 견적, 비교)
- **광고 소재(제목/설명)** → 클릭할 이유 제공 (AI 미리보기 = "다른 광고엔 없는 것")
- **확장소재** → AI 미리보기 기능 별도 강조

AI 미리보기는 검색어가 아니라 **클릭률(CTR)을 올리는 무기**다.
같은 순위에서도 이 문구 하나로 클릭률이 2~3배 차이날 수 있다.

### 소재 예시

| 위치 | 소재 예시 |
|------|---------|
| 제목1 | 인테리어 견적 한 번에 비교 |
| 제목2 | AI 완성 이미지 무료 제공 |
| 제목3 | 검증된 업체만 견적 모음 |
| 설명 | 여러 업체에 일일이 연락 No. AI가 완성 이미지 먼저 보여드리고, 견적은 한 번에 비교. 무료. |

### 확장소재 활용

네이버 확장소재(부가정보)에서 AI 미리보기를 별도로 강조:

```
[이미지형 확장소재]
AI 인테리어 미리보기 → 무료 체험

[텍스트 확장소재 - 추가설명]
· AI가 내 공간 인테리어 완성 이미지를 먼저 보여드립니다
· 마음에 들면 그때 견적 비교 → 부담 없는 프로세스
· 검증된 인테리어 업체 견적만 모아서 비교
```

## 5. 지역별 랜딩페이지 전략

> Q: URL 파라미터로 지역별 페이지 만들면, 지역마다 광고그룹을 따로 만들어야 하는 거 아닌가?
> 지역이 많아지면 관리가 너무 비효율적이지 않나?

**결론: 맞다. 지역별 접근은 비교 플랫폼에 맞지 않는다.**

### 왜 지역별 세분화가 우리에겐 비효율적인가

지역별 분리 전략은 **지역에만 서비스하는 단일 업체**에게 유효하다.

```
지역 인테리어 업체:
서울 업체 → 서울 손님만 필요 → "서울인테리어" 키워드 + 서울 전용 랜딩 = 효율적

우리 비교 플랫폼:
전국 서비스 → 어느 지역이든 견적 비교 가능 → 지역 쪼개기 = 오히려 분산
```

지역별로 광고그룹을 만들면:
- 관리 포인트가 수십 개로 증가
- 각 그룹별 데이터가 쌓이는 속도 느림 → 최적화 어려움
- 예산이 분산되어 어느 그룹도 충분한 노출 확보 못함

### 대신 이렇게 한다: 의도 기반 단일 랜딩 + 동적 지역명

전략: 광고그룹은 키워드 의도별로 나누되(위 그룹A/B/C), 랜딩페이지는 1개만 유지하되 **품질지수를 높이는 방향**으로 개선.

**지역 파라미터는 2~3개 주요 도시만 적용:**

```
일반 → yoursite.com (전국형 랜딩)
대구 광고그룹 → yoursite.com?region=daegu
서울 광고그룹 → yoursite.com?region=seoul
```

파라미터가 있으면 H1과 첫 문장에만 지역명이 바뀌는 방식:
```
없음: "인테리어 견적, 한 번에 비교해보세요"
대구: "대구 인테리어 견적, 한 번에 비교해보세요"
서울: "서울 인테리어 견적, 한 번에 비교해보세요"
```

이것만으로도 품질지수 개선 효과가 있고, 광고그룹은 2~3개만 추가 생성.

### 지역 광고그룹 우선순위 (테스트용 2~3개만)

```
대구/경북  → 처음 테스트한 지역, 데이터 있음
서울/경기  → 가장 큰 시장
부산/경남  → 3위 시장
```

나머지 지역은 그룹A(비교 의도 키워드)로 자연스럽게 유입됨.

## 6. 빠른 테스트 액션플랜

### Week 1: 기반 세팅

```
Day 1-2:
□ 자동입찰 → 수동입찰로 전환
□ 그룹A(비교 의도 키워드) 광고그룹 신규 생성
□ 광고 소재 2개 버전 작성 (AI 강조 vs 무료견적 강조)
□ 확장소재(부가정보) 최대한 채우기

Day 3-4:
□ 그룹B(지역+의도 2~3개) 추가 생성
□ 지역 파라미터 랜딩 개발 (H1만 바꾸는 방식, 1~2시간 작업)

Day 5-7:
□ 데이터 수집 (노출수, CTR, 클릭수)
□ 아직 조정 X — 최소 7일 데이터 필요
```

### Week 2: 최적화

```
□ CTR 낮은 소재 교체
□ 전환 없는 키워드 입찰가 낮추기
□ 전환 있는 키워드 입찰가 올리기
□ 품질지수 변화 확인
```

## 7. 입찰가 기준

| 그룹 | 키워드 예시 | 시작 입찰가 | 목표 |
|------|-----------|-----------|------|
| A (비교의도) | 인테리어견적비교 | 300~500원 | 1~3페이지 상단 진입 |
| B (지역+의도) | 대구아파트인테리어견적 | 200~400원 | 노출 확보 |
| C (레드오션) | 대구인테리어 | 보류 또는 최소값 | 품질 먼저 |

> 품질지수가 올라가면 **같은 입찰가로도 더 높은 순위**를 얻을 수 있다.
> 처음부터 입찰가를 높이는 건 낭비. 품질 먼저, 입찰가 조정은 그 다음.

| 지표 | 1주 목표 | 2주 목표 |

| 노출수 | 일 100+ | 일 500+ |

| 광고 순위 | 2페이지 이내 | 1페이지 진입 |

## 9. 요약

1. **지역 키워드보다 의도 키워드** — "인테리어견적비교" 계열이 우리 서비스에 맞음
2. **AI 미리보기는 키워드 X, 소재/확장소재에서 활용** — CTR 올리는 무기
3. **지역별 광고그룹 전체화는 비효율** — 주요 2~3개 도시만 선택적으로 적용
4. **자동입찰 버리고 수동 300~500원** — 지금 100원은 경쟁이 안 됨
5. **품질지수 먼저, 입찰가 확대는 그 다음** — 순서가 중요
