## 광고 편집 화면 전체 구조

피드 광고와 동일한 2개 영역을 채워야 합니다:

```
[영역 A] 광고 메인 편집 화면 (스크롤)
  ① 광고 이름
  ② 파트너십 광고
  ③ 대표 계정
  ④ 광고 설정 (형식)
  ⑤ 랜딩 페이지 + 웹사이트 URL + 표시 링크
  ⑥ 광고 크리에이티브 → [크리에이티브 설정] 클릭하면 영역 B가 열림
  ⑦ 크리에이티브 테스트
  ⑧ 언어
  ⑨ 추적 (URL 매개변수)

[영역 B] 크리에이티브 설정 팝업 (좌측 사이드바 5단계)
  B-1. 크리에이티브 설정 (소스 URL, 사이트 링크)
  B-2. 미디어 (이미지 업로드)
  B-3. 자르기 (크롭)
  B-4. 문구 (기본 문구 + 제목 + 행동 유도)
  B-5. 개선 사항 (AI 옵션 — 전부 끄기)
```
