# Instagram 광고 테스트 실전 플레이북

> 작성일: 2026-04-10
> 목적: 인스타그램 광고 처음인 사람이 이 문서만 보고 광고 집행 + A/B 테스트까지 완료할 수 있도록 작성
> 소재: 스토리 6종 + 피드 5종 = 총 11종 준비 완료

## 0. 시작 전 체크리스트 (10분)

광고를 만들기 전에 아래 3가지가 반드시 준비되어야 합니다.

| # | 준비물 | 상태 확인 | 없으면 |
|---|--------|---------|--------|
| 1 | **Instagram 비즈니스 계정** | 프로필 → 설정 → 계정 유형 → "비즈니스" 확인 | 개인 계정이면 무료로 전환 가능 (설정 → 계정 → 프로페셔널 계정으로 전환) |
| 2 | **Meta 비즈니스 스위트 연결** | [business.facebook.com](https://business.facebook.com) 접속 → 로그인 가능한지 확인 | Facebook 페이지 1개 만들기 → Instagram 계정 연결 |
| 3 | **결제 수단 등록** | Meta 광고 관리자 → 결제 설정 → 카드 등록 | 신용카드 또는 체크카드 등록 (최소 결제 단위: 1,000원) |

### Instagram 비즈니스 계정 전환 (1분)

이미 비즈니스 계정이면 건너뛰세요.

```
Instagram 앱 → 프로필 → 우측 상단 ≡ → 설정 및 개인정보
→ 계정 유형 및 도구 → 프로페셔널 계정으로 전환
→ 카테고리: "인테리어 디자인" 선택 → 비즈니스 선택 → 완료
```

### Facebook 페이지 연결 (5분)

Instagram 광고는 반드시 Facebook 페이지를 통해 집행합니다.

```
1. facebook.com → 좌측 메뉴 "페이지" → "새 페이지 만들기"
2. 페이지 이름: "인테리어 견적 비교" (또는 서비스명)
3. 카테고리: "인테리어 디자인 서비스"
4. 페이지 생성 완료
5. Instagram 앱 → 설정 → 비즈니스 → Facebook 페이지 연결 → 위 페이지 선택
```

## 2. 광고 테스트 전략 설계

### 핵심 원칙: "11개를 한꺼번에 돌리지 않는다"

> 초보가 가장 많이 하는 실수: 소재를 다 올려놓고 "Meta가 알아서 하겠지" → 예산 분산 → 학습 안 됨 → 돈만 날림

**우리의 전략: 3단계 깔때기 테스트**

```
[1단계] 소재 선별전 (Day 1~4)     — 11종 중 상위 4종 찾기
           ↓
[2단계] 승자 집중 운영 (Day 5~14)  — 상위 4종으로 본격 테스트
           ↓
[3단계] 스케일업 (Day 15~)        — 승자 1~2종에 예산 집중
```

### 예산 계획

| 단계 | 기간 | 일 예산 | 총 예산 |
|------|------|--------|--------|
| 1단계 소재 선별 | 4일 | 20,000원 | 80,000원 |
| 2단계 집중 테스트 | 10일 | 20,000원 | 200,000원 |
| 3단계 스케일업 | 이후 계속 | 30,000원~50,000원 | 월 예산에 맞춰 조절 |
| **합계** | **약 1개월** | — | **약 50만원** |

## 3. [1단계] 소재 선별전 — Day 1~4

### 목표
> 11개 소재 중 **CTR(클릭률) 상위 4개**를 골라내는 것. 이 단계에서는 전환(폼 제출)은 신경 쓰지 않는다. 오직 "어떤 이미지가 사람들의 시선을 잡는가"만 본다.

### 3-1. Meta 광고 관리자 접속

```
1. ads.manager.facebook.com 접속 (PC 권장, 모바일도 가능)
2. 또는 "Meta Business Suite" 앱 → 광고 → 광고 만들기
```

### 3-2. 캠페인 만들기 — 스텝 바이 스텝

#### Step 1: 캠페인 생성

```
광고 관리자 → [+ 만들기] 초록 버튼 클릭
```

| 설정 항목 | 선택할 값 | 이유 |
|----------|---------|------|
| 캠페인 목표 | **트래픽** | 전환 목표는 데이터가 쌓인 뒤에. 처음엔 트래픽이 학습이 빠름 |
| 캠페인 이름 | `인테리어_소재선별_Phase1` | 나중에 헷갈리지 않도록 |
| A/B 테스트 | OFF (끄기) | 우리가 직접 세트를 나눠서 비교할 것 |
| 어드밴티지+ 캠페인 예산 | **OFF (끄기)** | 각 광고 세트별로 예산을 따로 잡아야 공정 비교 가능 |

> **중요**: "어드밴티지+ 캠페인 예산"이 켜져 있으면 Meta가 멋대로 예산을 한쪽에 몰아버립니다. 반드시 끄세요.

### 3-3. 1단계 결과 확인 방법 (Day 4)

4일이 지나면 광고 관리자에서 데이터를 확인합니다.

```
광고 관리자 → 캠페인 선택 → "광고" 탭 클릭 → 열 편집
```

**반드시 봐야 할 지표 4가지:**

| 지표 | 어디서 보는가 | 의미 |
|------|-----------|------|
| **CTR (링크 클릭률)** | 열 편집 → "CTR(링크 클릭률)" 추가 | 광고를 본 사람 중 몇 %가 클릭했나 → **핵심 지표** |
| **CPC (링크 클릭당 비용)** | 열 편집 → "CPC(링크 클릭당 비용)" 추가 | 클릭 1번에 얼마 썼나 |
| **노출수** | 기본 표시됨 | 광고가 몇 번 보여졌나 (최소 1,000 이상이어야 판단 가능) |
| **링크 클릭수** | 기본 표시됨 | 실제 랜딩페이지로 넘어간 횟수 |

**판단 기준:**

| CTR | 등급 | 판정 |
|-----|------|------|
| **2.0% 이상** | S급 | 2단계 확정 진출 |
| **1.0~2.0%** | A급 | 2단계 후보 |
| **0.5~1.0%** | B급 | 탈락 (소재 교체 대상) |
| **0.5% 미만** | C급 | 즉시 중단 |

### 3-4. 1단계 완료 시 해야 할 일

```
1. 전체 11개 소재를 CTR 순으로 정렬
2. 상위 4개를 선정 (피드 2개 + 스토리 2개가 이상적)
3. 하위 7개 광고는 "끄기" (삭제X, 일시정지)
4. 상위 4개만 가지고 2단계로 진입
```

**예시 결과 시나리오:**

```
순위  코드   CTR    CPC     판정
─────────────────────────────────
1위   S4    2.3%   280원   ✅ 진출 (손실 공포가 가장 강력)
2위   F4    1.8%   350원   ✅ 진출
3위   S1    1.5%   400원   ✅ 진출
4위   F1    1.2%   450원   ✅ 진출
5위   F5    0.9%   520원   ❌ 탈락
...
```

## 4. [2단계] 승자 집중 운영 — Day 5~14

### 목표
> 1단계 상위 4개 소재로 **실제 전환(폼 제출)**이 발생하는지 검증. 동시에 타겟을 세분화해서 어떤 타겟이 가장 반응하는지 파악.

### 4-1. 캠페인 구조 변경

기존 1단계 캠페인은 일시정지하고, 새 캠페인을 만듭니다.

```
캠페인: "인테리어_전환테스트_Phase2"
├── 광고 세트 A: "코어타겟_신혼부부" (일 5,000원)
│   ├── 승자 소재 1
│   ├── 승자 소재 2
│   ├── 승자 소재 3
│   └── 승자 소재 4
├── 광고 세트 B: "코어타겟_30~40대_주택소유" (일 5,000원)
│   ├── 승자 소재 1~4 (동일)
├── 광고 세트 C: "코어타겟_인테리어관심" (일 5,000원)
│   ├── 승자 소재 1~4 (동일)
└── 광고 세트 D: "브로드타겟_연령만" (일 5,000원)
    ├── 승자 소재 1~4 (동일)
```

### 4-2. 타겟 세트 상세

| 세트 | 연령 | 관심사 | 의도 |
|------|------|--------|------|
| A: 신혼부부 | 25~35세 | 신혼, 웨딩, 이사, 인테리어 | 결혼 준비 중인 사람 |
| B: 주택소유자 | 35~50세 | 부동산, 아파트, 리모델링, 홈인테리어 | 기존 집 리모델링 |
| C: 인테리어 관심 | 28~45세 | 인테리어 디자인, 홈데코, 오늘의집, 집꾸미기 | 인테리어에 능동적 관심 |
| D: 브로드 | 28~50세 | 없음 (타겟팅 안 함) | Meta AI가 알아서 최적화하게 |

> **세트 D가 중요합니다.** 의외로 관심사를 안 넣은 "브로드 타겟"이 가장 성과가 좋은 경우가 많습니다. Meta의 AI가 학습할 여지를 주는 겁니다.

### 4-3. 캠페인 목표 변경

| 항목 | 1단계 | 2단계 (변경) |
|------|------|------------|
| 캠페인 목표 | 트래픽 | **전환** |
| 전환 이벤트 | 없음 | 랜딩페이지 방문 (또는 리드 제출) |
| 배치 | 수동 (피드/스토리 분리) | **어드밴티지+ 배치** (Meta가 알아서 피드/스토리/릴스 배분) |

> 2단계부터는 "어드밴티지+ 배치"로 바꿉니다. 1단계에서 소재별 성과를 이미 확인했으므로, 이제는 Meta AI에게 최적 배치를 맡겨도 됩니다.

### 4-4. 노출 위치 설정 (어드밴티지+ 배치)

```
광고 세트 편집 → 배치 → "어드밴티지+ 배치" 선택
→ 자동으로 Instagram 피드, 스토리, 릴스, 탐색 탭에 최적 배분
```

> 스토리용 9:16 이미지와 피드용 1:1 이미지를 **같은 광고에 함께 업로드** 가능합니다.
> 광고 만들기 → 미디어 추가 → "배치별 에셋 맞춤 설정" → 피드: 1:1 이미지 / 스토리: 9:16 이미지 등록

### 4-5. Meta 픽셀 설치 (전환 추적 — 매우 중요)

전환 목표 캠페인을 돌리려면 Meta 픽셀이 랜딩페이지에 설치되어 있어야 합니다.

```
1. Meta 이벤트 관리자 → 데이터 소스 → 픽셀 추가
2. 픽셀 이름: "인테리어 랜딩 픽셀"
3. 설치 방법: "코드 직접 추가" 선택
4. <head> 태그 안에 제공되는 코드 복붙
5. 표준 이벤트 추가:
   - 페이지 방문: fbq('track', 'PageView')  ← 기본
   - 견적 요청 제출: fbq('track', 'Lead')   ← 폼 제출 완료 시
```

**Next.js에 Meta 픽셀 넣는 위치:**
```
src/app/layout.tsx → <head> 안에 <Script> 태그로 삽입
폼 제출 완료 시 → fbq('track', 'Lead') 호출
```

> 픽셀을 못 넣겠으면 2단계에서도 "트래픽" 목표로 돌려도 됩니다. 다만 전환 목표가 더 효율적입니다.

### 4-6. 2단계 성과 판단 (Day 14)

| 지표 | 좋음 | 보통 | 나쁨 |
|------|------|------|------|
| **CTR (클릭률)** | 2% 이상 | 1~2% | 1% 미만 |
| **CPC (클릭당 비용)** | 300원 이하 | 300~600원 | 600원 이상 |
| **CPL (리드당 비용)** | 5,000원 이하 | 5,000~15,000원 | 15,000원 이상 |
| **랜딩→폼 전환율** | 5% 이상 | 2~5% | 2% 미만 |

**판단 의사결정 트리:**

```
CTR 높고 + 전환도 높음
→ 축하합니다! 3단계(스케일업)로 바로 이동

CTR 높은데 + 전환 낮음 (랜딩 이탈)
→ 소재는 OK. 랜딩페이지 개선 필요 (로딩 속도, CTA 위치, 폼 길이)

CTR 낮음 + 전환도 낮음
→ 타겟을 바꿔보기 (세트 A~D 중 성과 좋은 세트에 집중)
→ 그래도 안 되면 소재 전면 교체 (meta-ad-copy-all-patterns.md 참조)

특정 타겟 세트만 성과 좋음
→ 그 세트에 예산 몰아주기 (다른 세트 일시정지)
```

## 8. 소재 피로도 관리 (2주 후)

### 소재 피로도란?
같은 사람에게 같은 광고가 반복 노출되면 CTR이 떨어집니다. 이걸 "소재 피로도"라고 합니다.

### 피로도 감지 신호

```
□ 빈도(Frequency)가 3.0 이상 → 같은 사람에게 3번 이상 노출됨
□ CTR이 처음 대비 30% 이상 하락
□ CPC가 처음 대비 50% 이상 상승
```

**위 신호가 하나라도 보이면 → 소재 교체 시점**

### 교체 방법

1. 기존 승자 소재의 **카피만 교체** (이미지 유지)
   - `meta-ad-copy-all-patterns.md`에서 새 카피를 가져옴
   - 예: F4 이미지는 그대로 두고 텍스트만 "이 인테리어가 1,750만원?" → "비교견적 없었으면 두 배 냈을 겁니다"

2. 기존 승자와 **같은 유형의 새 이미지** 제작
   - `ad-production-guide.md`의 AI 프롬프트로 새 이미지 생성
   - 레이아웃/구도는 동일, 인테리어 스타일만 변경 (모던→내추럴, 화이트→우드톤)

3. 완전히 새로운 유형 시도
   - 1단계에서 탈락했던 유형 중 "아쉽게 탈락한 것" 재도전
   - 또는 새로운 패턴 (영상/릴스 소재 추가)

## 9. 흔한 실수 & 해결법

| # | 실수 | 왜 문제인가 | 해결 |
|---|------|-----------|------|
| 1 | "어드밴티지+ 예산"을 킨 채 소재 테스트 | Meta가 한쪽에만 예산 몰아서 공정 비교 불가 | 1단계에서는 반드시 OFF |
| 2 | 소재 1~2개만 올리고 테스트 | 비교 대상이 없어서 "좋은지 나쁜지" 판단 불가 | 최소 3개 이상 동시 테스트 |
| 3 | 하루 만에 성과 판단 | 노출 수백 건으로는 통계적 의미 없음 | 최소 3~4일, 소재당 1,000 노출 이상 |
| 4 | 성과 나쁜 소재를 "더 기다리면 나아지겠지" | Meta AI는 초반 학습 결과를 기반으로 배분 → 안 좋은 건 계속 안 좋음 | 3일 후 CTR 0.5% 미만이면 바로 끄기 |
| 5 | 예산을 한 번에 크게 올림 | 학습 리셋 → 하루이틀 효율 급감 | 3일마다 20~30%씩만 |
| 6 | UTM 파라미터 안 붙임 | 어떤 소재에서 전환이 왔는지 추적 불가 | 반드시 소재별 utm_content 다르게 |
| 7 | 댓글 관리 안 함 | 부정 댓글이 달리면 광고 효율 급감 | 매일 댓글 확인, 부정적이면 숨기기 |
| 8 | 랜딩페이지 속도 안 확인 | 3초 넘으면 유저 53% 이탈 | 모바일에서 직접 URL 열어서 체크 |

## 11. 광고 승인 거부 시 대처법

Meta에서 광고가 거부되는 경우가 있습니다. 당황하지 마세요.

| 거부 사유 | 우리 소재에서 해당 가능성 | 해결 |
|----------|---------------------|------|
| 텍스트 비율 초과 | F1(금색 3D 텍스트)이 해당될 수 있음 | 텍스트 면적을 줄인 버전으로 교체 |
| Before/After 과장 | 크게 해당 없음 (우리는 B/A 소재가 없음) | — |
| 가격 관련 기만적 표현 | "1,600만원에 끝냈다"가 보장으로 읽힐 수 있음 | 본문에 "실제 이용자 사례 기반" 문구 추가 |
| 랜딩페이지 불일치 | 광고 카피와 랜딩 내용이 다를 때 | 랜딩페이지에 "비교견적" 관련 내용이 있는지 확인 |

**거부 후 프로세스:**
```
1. 거부 사유 확인 (광고 관리자 → 해당 광고 → "거부됨" 클릭)
2. 사유에 맞게 수정 (이미지 교체 or 텍스트 수정)
3. "검토 요청" 클릭
4. 재검토 (보통 24시간 내)
```

## 부록: 자주 묻는 질문

### Q. Instagram에서 직접 "게시물 홍보"하면 안 되나요?
**비추천합니다.** Instagram 앱의 "홍보하기" 버튼은 세팅 옵션이 극히 제한적입니다. 반드시 **Meta 광고 관리자(ads.manager.facebook.com)**에서 만드세요. 소재별 비교, 타겟 세분화, UTM 추적이 전부 가능합니다.

### Q. 하루 예산 10,000원이면 너무 적지 않나요?
**1단계에서는 충분합니다.** 목적이 "어떤 소재가 되는지 찾기"이므로, 넓게 뿌리는 게 아니라 정확하게 비교하는 겁니다. 스케일은 3단계에서 합니다.

### Q. 피드랑 스토리 중 뭐가 더 좋은가요?
**정답은 없고, 그래서 테스트하는 겁니다.** 일반적으로:
- 스토리: CPC가 저렴, 노출량 높음, 전환율은 상대적으로 낮음
- 피드: CPC가 높지만, 체류시간이 길어 전환율이 높은 경향

### Q. 릴스(Reels) 광고는 안 하나요?
1~2단계에서는 **이미지 소재로 피드+스토리만 테스트**합니다. 릴스는 영상 소재가 필요해서 별도 제작이 필요합니다. 2단계에서 성과가 검증된 후, 승자 소재를 기반으로 5~15초 영상을 만들어 릴스 테스트를 추가하는 게 순서입니다.

### Q. 광고비는 어떻게 결제되나요?
Meta가 자동으로 등록된 카드에서 차감합니다. 결제 주기는 보통 일정 금액(약 50,000원)에 도달하거나 월말에 청구됩니다. 일 예산은 "하루에 이 이상은 안 쓴다"는 상한선이지, 정확히 그 금액이 나가는 건 아닙니다 (±20% 변동 가능).
