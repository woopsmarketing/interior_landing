## 5단계: 광고 만들기 (F1~F5 반복)

### 5-1. 광고 기본 설정 (메인 편집 화면)

| 항목 | 설정값 | 비고 |
|------|--------|------|
| 광고 이름 | F1: `F1_궁금증_1600만원` | F2~F5는 `feed-ad-setup-guide.md` 참조 |
| 파트너십 광고 | **해제** | |
| Facebook 페이지 | **모아홈테리어** | 자동 선택됨 |
| Instagram 프로필 | **moahome_interior** | 자동 선택됨 |
| 광고 설정 | **광고 만들기** | |
| 크리에이티브 소스 | **수동 업로드** | |
| 형식 | **단일 이미지 또는 동영상** | |
| 여러 광고주의 광고 | ✅ 체크 | 기본값 유지 |
| 랜딩 페이지 | **웹사이트** | |
| 웹사이트 URL | F1~F3: `https://moahome.kr/` / F4~F5: `https://moahome.kr/form` | A/B 테스트 |
| 표시 링크 | `moahome.kr` | |
| 행동 유도 버튼 | **없음** (크리에이티브 설정에서 CTA 지정) | |
| 언어 | **해제됨** | |
| 추적 - URL 매개변수 | 각 광고별 UTM (아래 참조) | |

### 5-2. 광고별 URL 매개변수 (UTM)

| 광고 | URL 매개변수 |
|------|-------------|
| F1 | `utm_source=instagram&utm_medium=feed&utm_campaign=creative_test&utm_content=f1_curiosity` |
| F2 | `utm_source=instagram&utm_medium=feed&utm_campaign=creative_test&utm_content=f2_hidden` |
| F3 | `utm_source=instagram&utm_medium=feed&utm_campaign=creative_test&utm_content=f3_gap` |
| F4 | `utm_source=instagram&utm_medium=feed&utm_campaign=creative_test&utm_content=f4_price_shock` |
| F5 | `utm_source=instagram&utm_medium=feed&utm_campaign=creative_test&utm_content=f5_comparison` |

### 5-3. 크리에이티브 설정 (팝업)

**"크리에이티브 설정" 버튼 클릭** → 5단계 사이드바 팝업 열림

각 광고별 이미지, 제목, CTA 등 상세 내용은 **`docs/feed-ad-setup-guide.md`** 참조

### 5-4. F2~F5 만드는 방법

1. F1 광고 완성 후 게시
2. F1 광고를 **복제** (광고 이름 우클릭 → 복제)
3. 복제된 광고에서 이름, 이미지, 제목, URL, UTM만 `feed-ad-setup-guide.md` 보고 교체
4. F2~F5까지 반복
