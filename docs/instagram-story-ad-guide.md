# Instagram Story 광고 소재 가이드 — 6가지 유형별 완성본

> 작성일: 2026-04-08
> 목적: 스토리 전용(9:16) 광고 소재 6종 제작 가이드
> 이미지 생성: Gemini / ChatGPT
> 핵심 원칙: **문구가 전부다. 이미지는 문구를 받쳐주는 배경이다.**

---

## Instagram 광고 배치 종류 — 한눈에 이해하기

### 피드 광고 (Feed Ad)

사용자가 인스타 메인 화면을 **아래로 스크롤**할 때, 일반 게시물처럼 보이면서 중간에 노출되는 광고.
좋아요, 댓글, 저장, 공유 같은 일반 게시물과 비슷한 반응을 받을 수 있어서 가장 무난하게 쓰인다.

| 항목 | 값 |
|------|-----|
| 권장 비율 | **4:5** (세로형, 피드에서 가장 넓은 면적) |
| 실무 권장 해상도 | **1080 x 1350px** |
| 추가 지원 | 1:1 (1080x1080), 1.91:1 (1080x566) |
| 지원 포맷 | 이미지 + 영상 |

### 스토리 광고 (Story Ad)

사용자가 다른 사람 스토리를 넘길 때 **중간에 끼어드는** 광고.
화면을 꽉 채우는 세로형 전체화면이라 시선 집중이 강하다. 대신 지나가는 속도가 빨라서 **첫 1~2초에 핵심 메시지**가 보여야 한다.

| 항목 | 값 |
|------|-----|
| 권장 비율 | **9:16** (세로 풀스크린) |
| 권장 해상도 | **1080 x 1920px** |
| 지원 포맷 | 이미지 + 영상 |

### 릴스 광고 (Reels Ad)

사용자가 릴스를 연속으로 볼 때 **중간에 들어가는** 광고.
릴스 특성상 짧고 빠르고 몰입형이라서, 짧은 영상 훅/전후 비교/공간 변화/후기형 영상이 잘 맞는다.

| 항목 | 값 |
|------|-----|
| 권장 비율 | **9:16** (세로형) |
| 권장 해상도 | **1080 x 1920px** |
| 지원 포맷 | 이미지 + 영상 (영상이 훨씬 효과적) |

### 세 줄 요약

```
피드 = 스크롤하다 보는 것  → 4:5 (1080x1350) → 이미지/영상
스토리 = 스토리 사이에 뜨는 것 → 9:16 (1080x1920) → 이미지/영상
릴스 = 릴스 사이에 뜨는 것  → 9:16 (1080x1920) → 영상 위주
```

> Meta 광고관리자에서 "자동 배치"를 선택하면 피드/스토리/릴스에 알아서 노출.
> 배치별로 소재를 따로 만들면 각 배치에 최적화된 경험을 줄 수 있다.

## 스토리 광고가 피드와 다른 점

| | 피드 광고 | 스토리 광고 |
|---|---|---|
| 체류 시간 | 스크롤하다 멈춤 (2~5초) | **자동으로 넘어감 (최대 15초)** |
| 화면 | 다른 게시물과 경쟁 | **풀스크린 독점** |
| 행동 | 클릭/좋아요 | **위로 스와이프 (또는 CTA 버튼 탭)** |
| 이미지 비율 | 4:5 (1080x1350) | **9:16 (1080x1920)** |
| 핵심 | 시선 멈추기 | **1초 안에 읽히게, 3초 안에 스와이프 욕구** |

**스토리에서 문구가 더 중요한 이유**: 풀스크린이라 텍스트가 유일한 정보 전달 수단. 피드는 이미지 아래 캡션이 있지만, 스토리는 **이미지 위 텍스트가 전부**.

## 공통 레이아웃 규칙

### 세이프 존 (반드시 지켜야 함)

```
┌─────────────────────┐
│  ⛔ 상단 250px       │ ← 프로필 아이콘, 시간 표시 (텍스트 넣지 마세요)
│                     │
│  ✅ 메인 영역        │ ← 핵심 문구 + 비주얼 (여기에 집중)
│  (1080 x 1320px)    │
│                     │
│  ⛔ 하단 350px       │ ← "자세히 알아보기" CTA 버튼 영역
│                     │    (텍스트 겹치면 가려짐)
└─────────────────────┘

전체: 1080 x 1920px (9:16)
```

**실수 많이 하는 부분**: 상단/하단에 중요 문구를 넣으면 UI에 가려져서 안 보임.

### 텍스트 배치 공식

```
┌─────────────────────┐
│  (상단 여백)          │
│                     │
│  ┌───────────────┐  │
│  │ 훅 카피 (1줄)   │  │  ← 가장 큰 글씨, 1초에 읽히는 한 마디
│  │ 40~52px Bold   │  │
│  └───────────────┘  │
│                     │
│  ┌───────────────┐  │
│  │ 서브 카피      │  │  ← 훅을 보충하는 1~2줄
│  │ 24~28px       │  │
│  └───────────────┘  │
│                     │
│  ┌───────────────┐  │
│  │ CTA 유도 문구  │  │  ← "위로 스와이프" 또는 화살표
│  │ 20px          │  │
│  └───────────────┘  │
│                     │
│  (하단 여백)          │
└─────────────────────┘
```

### 공통 디자인 원칙

| 항목 | 규칙 |
|------|------|
| 폰트 | Pretendard Bold (헤드라인), Medium (서브) |
| 메인 텍스트 색상 | 흰색 `#FFFFFF` (어두운 배경) 또는 다크 `#1A1A1A` (밝은 배경) |
| 강조 색상 | 오렌지 `#FF6B00` (금액, 핵심 키워드) |
| 텍스트 가독성 | 어두운 배경 위 흰 글씨 + 텍스트 그림자 또는 반투명 배경 박스 |
| 텍스트 비율 | 화면의 20% 이하 (Meta 정책) |
| 정렬 | 가운데 정렬 (스토리 특성상 중앙이 시선 집중) |

## 유형 1: 궁금증 유발 — 숨겨진 정보형

> **원리**: "나만 모르는 게 있다"는 불안감이 클릭을 만든다.
> 답을 절대 주지 않는다. 답은 랜딩페이지에 있다.

### 레이아웃

```
┌─────────────────────────┐
│  (상단 세이프존)           │
│                         │
│                         │
│    커튼 사이로 보이는      │
│    고급 인테리어 이미지     │
│    (어둡고 신비로운 분위기)  │
│                         │
│  ┌─────────────────┐    │
│  │                 │    │
│  │  "인테리어 업체가  │    │  ← 훅 카피 (흰색, 48px Bold)
│  │   절대 먼저       │    │
│  │   말 안 하는 것"  │    │
│  │                 │    │
│  └─────────────────┘    │
│                         │
│     확인하기 ↑           │  ← CTA 유도 (오렌지, 20px)
│                         │
│  (하단 세이프존)           │
└─────────────────────────┘
```

### 문구

```
훅 카피:
"인테리어 업체가
 절대 먼저 말 안 하는 것"

CTA 유도:
"확인하기 ↑"
```

**왜 이 문구인가**:
- "절대"라는 단어가 강한 호기심 유발
- "말 안 하는 것" → 뭔데? → 스와이프
- 서브 카피가 없는 게 핵심. **짧을수록 궁금증이 커진다**
- 피드용(A2)에서 "알려주지 않는 것"을 "말 안 하는 것"으로 바꿈 → 구어체라 스토리에 더 자연스러움

### 이미지 생성 프롬프트

```
어두운 분위기의 고급 인테리어 거실.
검은 커튼이 양쪽에서 살짝 열려 있고,
그 사이로 따뜻한 간접조명이 있는 모던 거실이 보인다.
신비롭고 비밀스러운 분위기.
사실적인 사진 스타일.
세로 9:16 비율 (1080x1920px).
하단 40%는 텍스트 오버레이용으로 어두운 그라데이션.
```

## 유형 2: 가격 충격형

> **원리**: 한국에서 가장 강력한 트리거는 돈이다.
> "이게 이 가격에?" 반응이 스와이프를 만든다.

### 레이아웃

```
┌─────────────────────────┐
│  (상단 세이프존)           │
│                         │
│  ┌─────────────────┐    │
│  │ "실제 시공 사례"   │    │  ← 태그 (반투명 검정 배경, 16px)
│  └─────────────────┘    │
│                         │
│    누가 봐도 고급스러운    │
│    대리석 주방 +          │
│    넓은 거실 인테리어      │
│                         │
│                         │
│  ┌─────────────────┐    │
│  │                 │    │
│  │  "이 인테리어     │    │  ← 훅 카피 (흰색, 48px Bold)
│  │   2천만원 안에    │    │     "2천만원" (오렌지, 52px Bold)
│  │   했다고?"       │    │
│  │                 │    │
│  │  비교견적 없었으면  │    │  ← 서브 카피 (흰색 70%, 22px)
│  │  4천만원          │    │     "4천만원" (오렌지)
│  │  냈을 겁니다      │    │
│  │                 │    │
│  └─────────────────┘    │
│                         │
│     내 견적 확인하기 ↑    │  ← CTA 유도
│                         │
│  (하단 세이프존)           │
└─────────────────────────┘
```

### 문구

```
태그:
"실제 시공 사례"

훅 카피:
"이 인테리어
 2천만원 안에 했다고?"

서브 카피:
"비교견적 없었으면
 4천만원 냈을 겁니다"

CTA 유도:
"내 견적 확인하기 ↑"
```

**왜 이 문구인가**:
- "2천만원"과 "4천만원"을 동시에 보여줘서 **갭 충격**을 만듦
- "했다고?" → 의문형이라 자연스럽게 "진짜?" 반응 유도
- "실제 시공 사례" 태그가 신뢰성 추가 (광고 같지 않게)
- 피드용(B1)과 같은 메시지지만 세로 풀스크린에서 숫자가 **훨씬 크게** 보여서 충격이 배가됨

### 이미지 생성 프롬프트

```
한국 아파트 30평대 거실+주방이 고급스럽게 완성된 사진.
대리석 아일랜드 식탁, 황동 팬던트 조명,
화이트+골드 톤의 모던 럭셔리 스타일.
따뜻한 간접조명, 큰 창으로 자연광.
인테리어 매거진 표지 스타일.
세로 9:16 비율 (1080x1920px).
하단 50%는 텍스트 오버레이용으로 검정→투명 그라데이션.
```

## 유형 3: 나란히 비교 + 절약 직격탄형

> **원리**: 숫자 비교는 뇌가 자동으로 처리한다.
> 두 숫자의 차이가 클수록 반응이 강하다.

### 레이아웃

```
┌─────────────────────────┐
│  (상단 세이프존)           │
│                         │
│                         │
│    고급 인테리어 배경      │
│    (밝고 넓은 거실)        │
│                         │
│  ┌──────┐  ┌──────┐    │
│  │A업체  │  │B업체  │    │  ← 비교 박스 2개
│  │3,600  │  │1,980  │    │     A: 흰 배경 + 빨간 취소선
│  │만원   │  │만원   │    │     B: 오렌지 배경 + 흰 글씨
│  └──────┘  └──────┘    │
│                         │
│  ┌─────────────────┐    │
│  │ "같은 30평인데     │    │  ← 훅 카피 (흰색, 44px Bold)
│  │  1,620만원        │    │     "1,620만원" (오렌지, 48px)
│  │  차이 납니다"      │    │
│  └─────────────────┘    │
│                         │
│    비교하면 답이 보입니다 ↑ │  ← CTA 유도
│                         │
│  (하단 세이프존)           │
└─────────────────────────┘
```

### 문구

```
비교 박스:
"A업체 3,600만원"  (취소선)
"B업체 1,980만원"  (오렌지 강조)

훅 카피:
"같은 30평인데
 1,620만원 차이 납니다"

CTA 유도:
"비교하면 답이 보입니다 ↑"
```

**왜 이 문구인가**:
- 피드(B2)에서는 "어디서 사라진 걸까요?"로 끝났는데, 스토리는 시간이 없으니 **답을 바로 줌**
- "1,620만원 차이"라는 구체적 숫자가 핵심 — 추상적인 "수백만원"보다 **5배 강력**
- A업체에 취소선을 넣으면 "비싼 곳 = 잘못된 선택" 느낌이 시각적으로 전달됨
- "같은 30평인데"가 중요 — 조건이 같다는 걸 먼저 알려줘야 가격 차이가 충격이 됨

### 이미지 생성 프롬프트

```
밝고 깨끗하게 완성된 한국 아파트 30평 거실.
화이트 벽면, 원목 바닥, 모던 미니멀 스타일.
자연광이 들어오는 환한 분위기.
사실적인 사진 스타일.
세로 9:16 비율 (1080x1920px).
중간~하단에 텍스트 오버레이를 넣을 수 있도록
하단 60%에 걸쳐 검정→투명 그라데이션.
```

## 유형 4: 손실 공포형

> **원리**: 사람은 100만원을 얻는 것보다 100만원을 잃는 것에 2배 더 강하게 반응한다.
> "당신은 지금 손해 보고 있다"가 핵심 메시지.

### 레이아웃

```
┌─────────────────────────┐
│  (상단 세이프존)           │
│                         │
│                         │
│    약간 어두운 톤의       │
│    인테리어 또는           │
│    공사 현장 사진          │
│    (불안한 분위기)         │
│                         │
│                         │
│  ┌─────────────────┐    │
│  │                 │    │
│  │  "비교 없이       │    │  ← 훅 카피 (흰색, 44px Bold)
│  │   계약했다가"     │    │
│  │                 │    │
│  │  1,200만원       │    │  ← 금액 (오렌지, 56px ExtraBold)
│  │                 │    │
│  │  "더 냈습니다"    │    │  ← 후속 (흰색, 44px Bold)
│  │                 │    │
│  └─────────────────┘    │
│                         │
│  ┌─────────────────┐    │
│  │ 견적 비교 한 번이면 │    │  ← 서브 (흰색 70%, 20px)
│  │ 막을 수 있었습니다 │    │
│  └─────────────────┘    │
│                         │
│     무료로 비교하기 ↑     │
│                         │
│  (하단 세이프존)           │
└─────────────────────────┘
```

### 문구

```
훅 카피:
"비교 없이 계약했다가"

금액 (초대형):
"1,200만원"

후속:
"더 냈습니다"

서브 카피:
"견적 비교 한 번이면
 막을 수 있었습니다"

CTA 유도:
"무료로 비교하기 ↑"
```

**왜 이 문구인가**:
- 핵심은 **문장을 쪼개는 것**. "비교 없이 계약했다가" → (시선 이동) → "1,200만원" → "더 냈습니다"
- 금액을 **중간에 단독으로 초대형**으로 넣어서, 스크롤 중에도 숫자가 먼저 눈에 꽂힘
- "더 냈습니다" → 과거형이라 **이미 일어난 일**처럼 느껴짐 (실제 사례 느낌)
- "막을 수 있었습니다" → 후회 자극 + "나는 아직 막을 수 있다" 행동 동기
- 피드용(C-1)에서는 "그냥 맡겼다가"였는데 "비교 없이 계약했다가"로 변경 — 더 구체적이고 행동(비교)을 명시

## 안전 영역 주의사항 (스토리 + 릴스 공통)

스토리와 릴스는 화면 위아래에 인스타 UI가 겹칠 수 있다.
**상단/하단/가장자리에는 중요한 텍스트, 로고, 버튼성 문구를 붙이지 않아야 한다.**

```
⛔ 상단 ~250px : 프로필 아이콘, 진행 바, 시간 표시
✅ 중앙 ~1320px : 핵심 문구 + 비주얼 (여기에 집중)
⛔ 하단 ~350px : "자세히 알아보기" CTA 버튼, 텍스트 입력 영역
```

> Meta도 9:16 스토리/릴스 광고에서 가장자리 영역을 비워두라고 공식 안내하고 있다.

## 피드 vs 스토리 vs 릴스 소재 운영 전략

### 현재 보유 소재

| 배치 | 소재 | 비율 | 상태 |
|------|------|------|------|
| 피드 | A1~B2 (5종) | **1536x1024 (3:2)** → 4:5 (1080x1350)로 리사이즈 필요 | 리사이즈 필요 |
| 스토리 | 6종 (이 문서) | 9:16 (1080x1920) | 제작 예정 |
| 릴스 | 미정 | 9:16 (1080x1920) 영상 | 추후 검토 |

### 운영 타임라인

```
1주차: 피드 소재 5종 + 스토리 소재 6종 동시 테스트
       피드: 일 17,000원 / 스토리: 일 18,000원

2주차: 각각 상위 2~3종만 남기고 예산 집중

3주차~: 피드 승자 + 스토리 승자 조합으로 "자동 배치" 전환
        Meta가 피드/스토리/릴스 중 효율 좋은 곳에 자동 배분

(선택) 릴스 광고: Before→After 변환 영상, 후기형 짧은 영상 등
       스토리 소재에서 성과 좋은 것을 영상으로 확장하는 전략 가능
```
