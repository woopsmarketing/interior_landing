# 유튜브 광고 소재 제작 가이드

인테리어 견적 비교 서비스 — 광고 영상 소재 기획 및 제작 스펙

---

## 소재 규격

| 포맷 | 해상도 | 비율 | 최대 용량 | 길이 |
|------|--------|------|-----------|------|
| 범퍼 광고 | 1920x1080 | 16:9 | 1GB | 6초 |
| 인스트림 광고 | 1920x1080 | 16:9 | 1GB | 15초+ |
| 쇼츠 (선택) | 1080x1920 | 9:16 | 1GB | 60초 이내 |

## 6초 범퍼 광고 (3가지 버전)

### 버전 1: 질문형

> 핵심 메시지: 변화에 대한 궁금증 자극

| 시간 | 화면 | 텍스트 오버레이 | 오디오 |
|------|------|-----------------|--------|
| 0-3초 | Before 인테리어 이미지 (낡은 공간) | **"이 집이 어떻게 바뀔까요?"** | 가벼운 BGM |
| 3-6초 | After 인테리어 이미지 (AI 생성, 완성된 공간) | **"무료 견적 비교 →"** + URL | BGM + 효과음 |

- 전환 효과: 심플 페이드 (0.3초)
- 텍스트: 흰색, 굵은 고딕, 화면 하단 1/3

### 버전 2: 숫자형

> 핵심 메시지: 비교의 필요성 강조

| 시간 | 화면 | 텍스트 오버레이 | 오디오 |
|------|------|-----------------|--------|
| 0-3초 | 견적서 1장 → 여러 장 비교 이미지 | **"인테리어 견적, 아직 하나만 받으세요?"** | 가벼운 BGM |
| 3-6초 | 서비스 화면 또는 완성 인테리어 | **"100건+ 비교 데이터 기반 무료 견적"** | BGM + 효과음 |

- 전환 효과: 슬라이드 또는 페이드 (0.3초)
- 숫자 `100건+` 강조: 오렌지색(#F97316), 나머지 흰색

### 버전 3: AI 강조

> 핵심 메시지: AI 미리보기 기능 어필

| 시간 | 화면 | 텍스트 오버레이 | 오디오 |
|------|------|-----------------|--------|
| 0-3초 | 스마트폰 화면에 AI 생성 인테리어 이미지 목업 | **"AI가 미리 보여주는"** | 가벼운 BGM |
| 3-6초 | AI 생성 인테리어 완성 이미지 (풀스크린) | **"내 집 인테리어 완성 모습"** + URL | BGM + 효과음 |

- 전환 효과: 줌인 (스마트폰 → 풀스크린)
- AI 키워드 강조: 오렌지색

## 15초 인스트림 광고 (2가지 스토리)

### 스토리 A: 문제 → 해결

> 전략: 공감 → 솔루션 → AI 차별점 → CTA

| 시간 | 화면 | 텍스트/나레이션 | 비고 |
|------|------|-----------------|------|
| 0-3초 | 인테리어 고민하는 사람 (머리 감싸는 모습) | **"인테리어 업체 고르기, 너무 어렵죠?"** | 공감 유발, 훅 |
| 3-6초 | 여러 견적서 비교 화면 | **"여러 업체 견적을 한 번에 비교하세요"** | 핵심 가치 전달 |
| 6-10초 | AI 생성 인테리어 이미지 (변환 과정 애니메이션) | **"AI가 완성 모습도 미리 보여드려요"** | 차별화 포인트 |
| 10-15초 | 서비스 로고 + CTA 버튼 | **"무료 견적 신청 →"** + URL | 전환 유도 |

- 5초 지점(건너뛰기 버튼 등장)까지 핵심 메시지 전달 완료
- 나레이션: 따뜻하고 신뢰감 있는 목소리 (남/여 모두 가능)

### 스토리 B: 후기형

> 전략: 사회적 증거 → 서비스 흐름 → AI → CTA

| 시간 | 화면 | 텍스트/나레이션 | 비고 |
|------|------|-----------------|------|
| 0-3초 | 완성된 인테리어 사진 (밝고 깔끔한 공간) | **"200만원 아꼈어요"** (큰 텍스트) | 임팩트 있는 오프닝 |
| 3-8초 | 서비스 흐름 3단계 아이콘 애니메이션 | **"1분 신청 → 업체 매칭 → 견적 비교"** | 간편함 강조 |
| 8-12초 | AI 미리보기 이미지 (Before → After) | **"AI 미리보기까지"** | 부가 가치 |
| 12-15초 | 서비스 로고 + CTA | **"지금 무료로 시작 →"** + URL | 전환 유도 |

- 첫 3초에 금액 절감 수치로 주의 확보
- 3-8초 구간이 가장 긴 구간 — 서비스 이해에 집중

## 제작 방법 (CapCut 기준)

### 준비물

1. **AI 인테리어 이미지 4~5장**
   - ChatGPT (DALL-E / gpt-image-1) 또는 Midjourney로 생성
   - Before 이미지 1~2장 + After 이미지 2~3장
   - 프롬프트 예시:
     ```
     Photorealistic interior design photo of a modern Korean apartment
     living room, warm wood tones, natural light through large windows,
     minimalist furniture, wide-angle shot, architectural digest style
     ```

2. **텍스트 카피** — 위 스토리보드 참고
3. **서비스 로고** — PNG (투명 배경)
4. **BGM** — 밝고 가벼운 배경음악

### CapCut 제작 순서

1. CapCut 열기 → **새 프로젝트** → **16:9 (1920x1080)** 선택
2. 이미지를 타임라인에 순서대로 배치
3. 각 프레임 길이 조정:
   - 범퍼: 프레임당 2.7~3초 (전환 시간 고려)
   - 인스트림: 스토리보드 시간 배분 참고
4. **텍스트 오버레이 추가**:
   - 폰트: 프리텐다드, 노토산스 등 깔끔한 고딕체
   - 크기: 화면 너비의 60~80% 차지
   - 색상: 흰색 기본, 강조 단어는 오렌지(#F97316)
   - 위치: 화면 하단 1/3 또는 중앙
   - 그림자 또는 반투명 배경 박스 추가 (가독성)
5. **전환 효과 적용**:
   - 프레임 간: 심플 페이드 (0.3초)
   - 과도한 효과 사용 금지 (전문성 저하)
6. **BGM 추가**:
   - CapCut 내장 무료 음악 또는 저작권 프리 음원
   - 볼륨: 나레이션 있으면 20~30%, 없으면 50~60%
7. **내보내기**:
   - 해상도: 1080p (1920x1080)
   - 프레임: 30fps
   - 포맷: MP4
   - 품질: 높음

## 썸네일 가이드

유튜브에 영상 업로드 시 커스텀 썸네일 필수 설정.

### 사양

| 항목 | 값 |
|------|-----|
| 크기 | 1280x720px |
| 비율 | 16:9 |
| 포맷 | JPG 또는 PNG |
| 최대 용량 | 2MB |

### 디자인 원칙

- **텍스트는 3단어 이내**: "견적비교 무료", "200만원 절약", "AI 미리보기"
- **큰 폰트**: 모바일에서도 읽힐 수 있는 크기
- **밝은 이미지**: 어두운 썸네일은 CTR 저하
- **대비 강조**: 텍스트와 배경 색상 대비 확보
- **공간 사진 포함**: 완성된 인테리어 이미지가 CTR 향상에 효과적
- **얼굴 포함 시**: 사람 얼굴이 있으면 CTR 추가 향상 (선택)

### 썸네일 예시 구성

```
┌──────────────────────────────┐
│                              │
│   [완성된 인테리어 사진]       │
│                              │
│        ┌────────────┐        │
│        │ 무료 견적   │        │
│        │ 비교 →     │        │
│        └────────────┘        │
│                    [로고]     │
└──────────────────────────────┘
```

## A/B 테스트 계획

### 1차 테스트 (1~2주)

| 테스트 항목 | A | B |
|-------------|---|---|
| 범퍼 소재 | 버전 1 (질문형) | 버전 2 (숫자형) |
| 인스트림 소재 | 스토리 A (문제→해결) | 스토리 B (후기형) |

### 측정 지표

| 지표 | 범퍼 기준 | 인스트림 기준 |
|------|-----------|---------------|
| 조회율 (VTR) | - | 15%+ 목표 |
| 클릭율 (CTR) | 0.5%+ | 1%+ |
| 전환율 | - | 랜딩페이지 폼 제출 기준 |
| CPV | - | 50원 이하 목표 |
| CPM | 4,000원 이하 | - |

### 2차 테스트

- 1차에서 성과 좋은 소재 유지
- 승리한 소재의 변형 버전 추가 테스트
- 타겟팅 조합 테스트 (맞춤 의도 vs 관심분야)
