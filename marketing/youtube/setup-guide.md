# 유튜브 광고 설정 가이드

인테리어 견적 비교 서비스 — 유튜브 광고 캠페인 세팅 A to Z

---

## 1. 유튜브 채널 생성 (브랜드 계정)

### 1-1. 브랜드 계정으로 채널 만들기

1. [youtube.com](https://youtube.com) 접속 → 구글 계정 로그인
2. 우측 상단 프로필 → **설정** → **새 채널 만들기**
3. **브랜드 계정 사용** 선택 (개인 계정 X)
4. 채널명 입력: `인테리어 견적비교` 또는 서비스 브랜드명

### 1-2. 채널 브랜딩 설정

| 항목 | 권장 사양 | 내용 |
|------|-----------|------|
| 프로필 이미지 | 800x800px, PNG | 서비스 로고 또는 심볼 |
| 배너 이미지 | 2560x1440px, PNG | Before/After 인테리어 이미지 + 서비스 슬로건 |
| 워터마크 | 150x150px, PNG | 투명 배경 로고 |

### 1-3. 채널 설명 작성

```
인테리어 견적, 혼자 알아보지 마세요.
검증된 인테리어 업체의 견적을 한 번에 비교하고,
AI가 완성 모습까지 미리 보여드립니다.

무료 견적 신청: [랜딩페이지 URL]
```

- **링크 섹션**에 랜딩페이지 URL 추가
- **연락처 이메일** 설정 (비즈니스 문의용)

## 2. 구글 광고 계정 설정

### 2-1. 광고 계정 생성

1. [ads.google.com](https://ads.google.com) 접속
2. **새 Google Ads 계정** 생성
3. **전문가 모드로 전환** (스마트 캠페인 X — 세부 설정 불가)
4. 비즈니스 정보 입력: 업종 `인테리어/리모델링`, 국가 `대한민국`, 시간대 `서울`

### 2-2. 결제 수단 등록

1. **도구 및 설정** → **결제** → **결제 수단**
2. 신용카드 또는 체크카드 등록
3. 결제 프로필 유형: **사업자** (세금계산서 필요 시) 또는 **개인**

### 2-3. 유튜브 채널 연결

1. **도구 및 설정** → **설정** → **연결된 계정**
2. **YouTube** 선택 → **채널 추가**
3. 채널 URL 입력 → 인증 완료
4. 연결 완료 후 채널 동영상을 광고 소재로 사용 가능

## 3. 캠페인 생성

### 3-1. 캠페인 목표 설정

| 목표 | 설명 | 추천 상황 |
|------|------|-----------|
| **리드** | 폼 제출(견적 신청) 극대화 | 전환 데이터 축적 후 (50건+) |
| **웹사이트 트래픽** | 랜딩페이지 유입 극대화 | 초기 테스트 단계 |

- 초기에는 **웹사이트 트래픽** 목표로 시작
- 전환 데이터 50건 이상 쌓이면 **리드** 목표로 전환

### 3-2. 캠페인 유형

- **동영상** 캠페인 선택

### 3-3. 하위 유형 선택

| 하위 유형 | 길이 | 특징 | 용도 |
|-----------|------|------|------|
| **범퍼 광고** | 6초 | 건너뛰기 불가, CPM 과금 | 브랜드 인지도, 리타겟팅 |
| **건너뛸 수 있는 인스트림** | 15초+ | 5초 후 건너뛰기 가능, CPV 과금 | 서비스 설명, 전환 유도 |

- **추천**: 두 가지를 별도 캠페인으로 동시 운영
- 범퍼 = 인지도, 인스트림 = 전환 유도 역할 분담

## 4. 타겟팅 설정

### 4-1. 맞춤 의도 타겟팅 (Custom Intent Audiences)

구글에서 아래 키워드를 **최근 검색한 사용자**에게 광고 노출:

```
인테리어 견적
인테리어 비용
인테리어 업체 추천
리모델링 업체
아파트 인테리어
신혼집 인테리어
올수리 비용
인테리어 비교
인테리어 후기
셀프 인테리어
```

설정 방법:
1. 광고그룹 → **잠재고객** → **맞춤 세그먼트** → **새로 만들기**
2. **Google에서 이러한 검색어를 검색한 사용자** 선택
3. 위 키워드 입력 → 저장

### 4-2. 관심분야 타겟팅

| 카테고리 | 세부 항목 |
|----------|-----------|
| 주거/부동산 | 부동산 구매자, 주택 소유자 |
| 인테리어/홈데코 | 인테리어 디자인, 홈 인테리어 |
| 생활 이벤트 | 최근 이사, 신혼 |

### 4-3. 인구통계 설정

| 항목 | 설정값 |
|------|--------|
| 연령 | 25-54세 |
| 성별 | 전체 (초기), 이후 데이터 기반 조정 |
| 가구 소득 | 상위 50% (선택) |

### 4-4. 지역 타겟팅

| 우선순위 | 지역 | 입찰 조정 |
|----------|------|-----------|
| 1순위 | 서울 | +20% |
| 2순위 | 경기/인천 | 기본 |
| 3순위 | 광역시 (부산, 대구, 대전, 광주) | 기본 |

- 초기에는 **서울/수도권** 집중 → 데이터 확인 후 확대
- 지역별 성과 차이가 크면 별도 캠페인 분리

## 5. 입찰 전략

### 5-1. 범퍼 광고 (6초)

| 항목 | 설정값 |
|------|--------|
| 과금 방식 | **CPM** (1,000회 노출당 비용) |
| 입찰 전략 | 타겟 CPM |
| 시작 CPM | 3,000~5,000원 |
| 일예산 | 10,000~15,000원 |

### 5-2. 인스트림 광고 (15초+)

| 항목 | 설정값 |
|------|--------|
| 과금 방식 | **CPV** (30초 시청 또는 클릭 시 과금) |
| 입찰 전략 | 최대 CPV |
| 시작 CPV | 30~80원 |
| 일예산 | 10,000~20,000원 |

### 5-3. 예산 가이드

| 단계 | 일예산 | 월예산 (약) | 목적 |
|------|--------|-------------|------|
| 테스트 (1~2주) | 1~2만원 | 30~60만원 | 소재/타겟 테스트 |
| 최적화 (3~4주) | 2~5만원 | 60~150만원 | 성과 좋은 조합 스케일업 |
| 스케일 | 5만원+ | 150만원+ | 전환 극대화 |

## 6. GA4 연동

### 6-1. 구글 광고 ↔ GA4 연결

1. **GA4** → 관리 → **Google Ads 링크** → **연결**
2. 구글 광고 계정 선택 → 연결 확인
3. **구글 광고** 측에서도: 도구 및 설정 → 연결된 계정 → Google Analytics(GA4) 연결

### 6-2. 연동 후 확인사항

- GA4에서 구글 광고 캠페인별 유입/행동 데이터 확인 가능
- 구글 광고에서 GA4 전환 데이터 가져오기 가능
- 리마케팅 잠재고객 공유 가능

## 7. 전환 액션 설정

### 7-1. 전환 이벤트 정의

랜딩페이지에서 폼 제출 시 발생하는 이벤트를 전환으로 등록:

| 전환 액션 | 이벤트명 | 트리거 |
|-----------|----------|--------|
| 견적 신청 완료 | `form_submit` | Step 5 폼 제출 완료 시 |
| CTA 클릭 (보조) | `cta_click` | 견적 신청 버튼 클릭 시 |

### 7-2. 구글 광고에서 전환 설정

1. **도구 및 설정** → **측정** → **전환**
2. **새 전환 액션** → **가져오기** → **Google 애널리틱스(GA4) 속성**
3. `form_submit` 이벤트 선택 → **가져오기 및 계속**
4. 전환 창: 30일 (클릭 후), 1일 (조회 후)

### 7-3. 전환 추적 확인

- 테스트 폼 제출 → GA4 실시간 이벤트에서 `form_submit` 확인
- 구글 광고 → 전환 → 상태가 **전환 기록 중**인지 확인
- 24~48시간 내 데이터 반영
