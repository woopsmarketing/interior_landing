# 네이버 검색광고 키워드 목록

> 대상 서비스: AI 인테리어 견적 비교
> 키워드 도구: [네이버 키워드 플래너](https://searchad.naver.com) > 도구 > 키워드 도구
> 검색량 데이터: 실 집행 전 키워드 도구에서 최신 수치 확인 필수

---

## 그룹 1: 높은 전환 의도

견적/비교를 직접 검색하는 사용자. 전환 가능성이 가장 높다.
입찰가를 높게 설정하고 정확 매칭 위주로 운영한다.

| 키워드 | 예상 월간 검색량 | 권장 입찰가 | 우선순위 |
|--------|----------------|-----------|---------|
| 인테리어 견적 | 8,000~12,000 | 1,000~1,500원 | ★★★ |
| 인테리어 비교견적 | 1,000~2,000 | 1,200~1,800원 | ★★★ |
| 인테리어 업체 비교 | 3,000~5,000 | 1,000~1,500원 | ★★★ |
| 인테리어 견적 비교 | 2,000~4,000 | 1,200~1,800원 | ★★★ |
| 아파트 인테리어 견적 | 3,000~5,000 | 1,000~1,500원 | ★★★ |
| 원룸 인테리어 견적 | 1,500~3,000 | 800~1,200원 | ★★☆ |
| 인테리어 무료견적 | 2,000~4,000 | 1,000~1,500원 | ★★★ |
| 리모델링 견적 | 5,000~8,000 | 1,000~1,500원 | ★★★ |

**운영 가이드**:
- 매칭 타입: 정확 매칭 우선, 성과 확인 후 구문 매칭 확장
- 입찰가: 상위 3위 이내 노출 목표
- 모바일 가중치: +20~30%

---

## 그룹 2: 중간 전환 의도

인테리어를 알아보는 단계의 사용자. 정보 탐색 중이지만 전환 가능성 있다.
입찰가를 중간 수준으로 설정하고 소재로 전환 유도한다.

| 키워드 | 예상 월간 검색량 | 권장 입찰가 | 우선순위 |
|--------|----------------|-----------|---------|
| 인테리어 비용 | 15,000~25,000 | 600~1,000원 | ★★☆ |
| 인테리어 가격 | 10,000~18,000 | 600~1,000원 | ★★☆ |
| 집 인테리어 | 20,000~35,000 | 400~700원 | ★★☆ |
| 아파트 리모델링 | 8,000~15,000 | 700~1,200원 | ★★☆ |
| 인테리어 추천 | 12,000~20,000 | 500~800원 | ★★☆ |
| 인테리어 업체 추천 | 5,000~10,000 | 700~1,000원 | ★★☆ |
| 셀프 인테리어 vs 업체 | 500~1,500 | 400~700원 | ★☆☆ |

**운영 가이드**:
- 매칭 타입: 구문 매칭 또는 확장 매칭
- 소재 전략: 가격 비교/무료 견적 강조하여 전환 유도
- 전환률이 낮은 키워드는 2주 후 중지 검토

## 그룹 3: 지역 키워드

지역명+인테리어를 검색하는 사용자. 실제 시공 의향이 높다.
서비스 가능 지역에 한정하여 운영한다.

| 키워드 | 예상 월간 검색량 | 권장 입찰가 | 우선순위 |
|--------|----------------|-----------|---------|
| 서울 인테리어 | 8,000~15,000 | 800~1,200원 | ★★★ |
| 강남 인테리어 | 3,000~6,000 | 1,000~1,500원 | ★★☆ |
| 수원 인테리어 | 3,000~5,000 | 700~1,000원 | ★★☆ |
| 분당 인테리어 | 2,000~4,000 | 800~1,200원 | ★★☆ |
| 일산 인테리어 | 2,000~4,000 | 700~1,000원 | ★★☆ |
| 인천 인테리어 | 4,000~7,000 | 700~1,000원 | ★★☆ |
| 부산 인테리어 | 5,000~8,000 | 700~1,000원 | ★★☆ |

**운영 가이드**:
- 지역 타겟팅과 병행: 해당 지역 사용자에게만 노출
- 확장 시 추가 지역: 대구, 대전, 광주, 성남, 용인, 화성 등
- 소재에 "내 지역 인테리어 업체 비교" 문구 활용

| 인테리어 타일 | 자재 구매 목적 |

- 검색어 보고서 경로: 광고관리 > 보고서 > 검색어 보고서

## 키워드 확장 전략

초기 운영 데이터 축적 후 아래 방향으로 키워드를 확장한다.

### 공간 유형별 확장
- 빌라 인테리어 견적
- 오피스텔 인테리어
- 신축 아파트 인테리어
- 구축 아파트 리모델링

### 공간별 확장
- 주방 리모델링 견적
- 욕실 리모델링 비용
- 거실 인테리어 비용

### 시즌 키워드
- 이사 인테리어 (봄/가을 이사 시즌)
- 신혼집 인테리어 (웨딩 시즌)
- 입주 청소 + 인테리어

## 참고: 검색량 확인 방법

1. [네이버 검색광고](https://searchad.naver.com) 로그인
2. 도구 > 키워드 도구
3. 키워드 입력 후 조회
4. 월간 검색수(PC/모바일), 월평균 클릭수, 경쟁 정도 확인
5. 위 표의 예상 검색량은 참고용이며, 실제 수치는 시기와 트렌드에 따라 변동

> 위 검색량과 입찰가는 추정치이다. 반드시 키워드 도구에서 최신 데이터를 확인한 후 입찰가를 결정한다.
