# 네이버 검색광고 세팅 가이드

> 대상 서비스: AI 인테리어 견적 비교 랜딩페이지
> 랜딩 URL: `https://your-domain.vercel.app`
> 광고 플랫폼: [네이버 검색광고](https://searchad.naver.com)

---

## 1. 계정 개설

1. [searchad.naver.com](https://searchad.naver.com) 접속
2. 네이버 아이디로 로그인 (없으면 회원가입)
3. 광고주 유형 선택
   - **사업자**: 사업자등록번호 입력, 세금계산서 발행 가능
   - **개인**: 주민등록번호 인증, 소규모 테스트 시 적합
4. 광고주 정보 입력 완료 후 계정 활성화

> 참고: 사업자 계정이 세금 처리에 유리하므로 장기 운영 시 사업자 계정 권장

---

## 2. 광고비 충전

- **충전 방식**: 신용카드 / 계좌이체 / 무통장입금
- **최소 충전 권장액**: 10만원
  - 초기 테스트 기간(1~2주) 동안 키워드별 성과 데이터 수집 목적
  - 일예산 1~2만원 기준 약 5~10일 운영 가능
- **자동충전 설정**: 잔액 일정 금액 이하 시 자동 충전 (운영 안정성 확보)

---

## 3. 캠페인 생성

1. 광고관리 > 캠페인 만들기
2. **캠페인 유형**: 파워링크 (검색 결과 상단/하단 텍스트 광고)
3. **캠페인명**: `인테리어견적비교_파워링크_YYYYMM` (날짜 포함 권장)
4. **일예산 설정**:
   - 초기 테스트: 1~2만원/일
   - 안정화 후: 데이터 기반 조정
5. **노출 기간**: 시작일 설정, 종료일은 미설정 (수동 관리)

---

## 4. 광고그룹 생성

키워드 그룹별로 광고그룹을 분리하여 생성한다.

### 광고그룹 구성 예시

| 광고그룹명 | 키워드 유형 | 기본 입찰가 |
|-----------|-----------|-----------|
| GRP1_전환의도_높음 | 견적/비교 키워드 | 800~1,500원 |
| GRP2_전환의도_중간 | 비용/추천 키워드 | 400~800원 |
| GRP3_지역키워드 | 지역+인테리어 | 500~1,000원 |

### 타겟 설정

- **디바이스**: 모바일 우선 (모바일 입찰가 가중치 +20~30%)
  - 랜딩페이지가 모바일 최적화되어 있으므로 모바일 전환률이 높을 것으로 예상
- **지역 타겟팅**: 전국 또는 서비스 가능 지역으로 한정
  - 초기에는 수도권(서울/경기/인천) 집중 후 확대 권장
- **시간대**: 전체 시간 노출 (데이터 축적 후 성과 낮은 시간대 제외)

---

## 5. 키워드 등록

- 키워드 목록은 `keywords.md` 참고
- 키워드 도구(네이버 키워드 플래너)로 검색량 재확인
- **매칭 타입**:
  - 핵심 키워드: 정확 매칭으로 시작
  - 보조 키워드: 구문/확장 매칭 활용
- 등록 후 예상 순위/입찰가 확인하여 조정

---

## 6. 소재 등록

- 광고 텍스트 소재는 `ad-copy.md` 참고
- 광고그룹당 소재 2~3개 등록하여 A/B 테스트
- 소재별 성과(CTR, 전환률) 비교 후 하위 소재 교체

### 소재 작성 규칙 (네이버 가이드라인)

- 제목: 최대 15자 (공백 포함)
- 설명: 최대 45자 (줄당 25자 기준, 공백 포함)
- 과장/허위 표현 금지
- 최상급 표현("최고", "1등") 사용 시 객관적 근거 필요
- 특수문자 과다 사용 금지
- 랜딩 URL과 광고 내용 일치 필수

확장소재는 광고 클릭률을 높이는 부가 정보이다.

| 이용 후기 | `https://your-domain.vercel.app#reviews` |

### 추가설명

- "100건+ 무료 견적 비교"
- "AI 완성 이미지 미리보기"
- "1분 간편 신청"

### 전화번호 확장 (선택)

- 모바일에서 바로 전화 연결 가능

## 8. 전환추적 스크립트 설치

> 상세 설치 방법은 `tracking/setup-guide.md` 참고 (별도 작성 예정)

### 기본 설정

1. 네이버 검색광고 > 도구 > 전환추적 > 전환추적 스크립트 발급
2. 공통 스크립트: 모든 페이지 `<head>`에 삽입
3. 전환 스크립트: 폼 제출 완료 시점에 실행
   - 전환 유형: "신청완료" (리드 수집)
   - 전환 페이지: 폼 제출 성공 후 완료 화면

### 전환 이벤트 정의

| 전환 이름 | 트리거 시점 | 가치 |
|----------|-----------|------|
| 견적신청완료 | Step 5 제출 성공 | 주요 전환 |
| 폼시작 | Step 1 진입 | 보조 전환 (마이크로) |

## 9. 입찰 전략

### Phase 1: 데이터 수집기 (1~2주)

- **방식**: 수동 CPC
- **목표**: 키워드별 CTR, 전환률 데이터 확보
- **입찰가**: 키워드 도구 권장가의 80~100% 수준에서 시작
- **조정 주기**: 2~3일마다 성과 확인 후 입찰가 조정

### Phase 2: 최적화기 (3~4주)

- **방식**: 수동 CPC 유지, 성과 기반 조정
- **작업**:
  - 전환 없는 키워드 입찰가 하향 또는 중지
  - 전환 키워드 입찰가 상향
  - 소재 A/B 테스트 결과 반영
  - 제외 키워드 추가 (전환 무관 검색어 차단)

### Phase 3: 자동화 전환 (5주~)

- **방식**: 타겟 CPA (전환당 비용 목표)
- **전제 조건**: 최소 15~20건 이상 전환 데이터 축적
- **타겟 CPA**: Phase 2의 평균 CPA 기준 설정
- **주의**: 자동 입찰 전환 후 1~2주는 학습 기간, 급격한 변경 자제

## 10. 성과 모니터링

### 핵심 KPI 목표

| 지표 | 목표 | 비고 |
|------|------|------|
| CTR (클릭률) | 2% 이상 | 업종 평균 1.5~2.5% |
| CPC (클릭당 비용) | 500원 이하 | 키워드 그룹에 따라 차이 |
| 전환률 | 5% 이상 | 랜딩페이지 최적화 시 |
| CPA (전환당 비용) | 10,000원 이하 | 초기 목표, 데이터 기반 조정 |

### 모니터링 루틴

- **매일**: 소진 금액, 클릭수, 전환수 확인
- **주간**: 키워드별/소재별 성과 비교, 입찰가 조정
- **월간**: 전체 ROI 분석, 키워드 확장/축소 판단, 예산 재배분

### 성과 개선 체크리스트

- [ ] CTR 낮은 소재 교체
- [ ] 전환 없는 키워드 중지 또는 입찰가 하향
- [ ] 검색어 보고서에서 제외 키워드 추가
- [ ] 모바일/PC 성과 비교 후 디바이스 가중치 조정
- [ ] 시간대별 성과 확인 후 노출 시간 조정
- [ ] 랜딩페이지 이탈률 확인 후 페이지 개선

## 주의사항

- **광고 심사**: 소재 등록 후 심사에 1~2 영업일 소요. 주말/공휴일 제외.
- **소재 가이드라인**: 네이버 광고 정책 위반 시 비승인. 과장/허위/비교광고 주의.
- **랜딩페이지 일치**: 광고 내용과 랜딩페이지 내용이 불일치하면 품질지수 하락.
- **품질지수**: 키워드 품질지수(1~10)가 입찰가와 노출 순위에 영향. CTR과 랜딩 관련성이 핵심.
- **예산 소진**: 일예산 소진 시 당일 노출 중단. 핵심 시간대에 예산이 남도록 관리.
- **계절성**: 인테리어 수요는 봄(3~5월), 가을(9~11월)에 높음. 시즌에 예산 증액 고려.
