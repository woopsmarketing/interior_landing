| **설명 2줄** | 1분 만에 신청 완료. | 10자 |
| **표시 URL** | your-domain.vercel.app | - |

**타겟 키워드**: 인테리어 견적, 인테리어 비교견적, 인테리어 업체 비교, 인테리어 무료견적

## A/B 테스트 운영 방법

### 테스트 설계

1. 광고그룹당 소재 2~3개를 동시 등록
2. 네이버가 자동으로 노출을 분배 (균등 노출 설정 권장)
3. 최소 1~2주, 클릭 100회 이상 데이터 확보 후 판단

### 판단 기준

| 지표 | 우수 기준 | 조치 |
|------|---------|------|
| CTR | 2% 이상 | 유지 및 예산 집중 |
| CTR | 1% 미만 | 소재 교체 |
| 전환률 | 5% 이상 | 유지 |
| 전환률 | 2% 미만 | 소재 또는 랜딩페이지 개선 |

### 소재 교체 주기

- 2주 단위로 성과 리뷰
- 하위 성과 소재를 새 소재로 교체
- 동시에 2개 이상 변경하지 않음 (변수 통제)

## 네이버 소재 가이드라인 준수사항

### 승인 거부 주요 사유

| 위반 유형 | 예시 | 수정 방향 |
|----------|------|----------|
| 과장 표현 | "최저가 보장", "업계 1위" | 객관적 근거 없으면 삭제 |
| 허위 표현 | "100% 만족 보장" | 구체적 수치로 변경 |
| 비교 광고 | "A사보다 저렴" | 경쟁사 언급 삭제 |
| 특수문자 남용 | "★무료★견적★" | 특수문자 최소화 |
| 랜딩 불일치 | 광고에 "무료" 언급, 랜딩에 유료 | 광고-랜딩 내용 일치시키기 |
| 연락처 노출 | 설명에 전화번호 직접 기재 | 전화번호 확장소재 사용 |

### 작성 시 체크리스트

- [ ] 제목 15자 이내 (공백 포함)
- [ ] 설명 줄당 25자 이내 (공백 포함)
- [ ] 과장/허위/최상급 표현 없음
- [ ] 경쟁사 직접 비교 없음
- [ ] 특수문자 적절하게 사용 (1~2개 이내)
- [ ] 광고 내용과 랜딩페이지 내용 일치
- [ ] 맞춤법 검수 완료
- [ ] 모바일에서 잘림 없이 표시되는지 확인

## 소재 확장 아이디어

초기 3종 테스트 후, 성과에 따라 아래 방향으로 소재를 추가 제작한다.

### 시즌별 소재
- 봄: "봄맞이 인테리어, 견적부터 비교하세요"
- 가을: "이사철 인테리어, 업체 비교로 시작"
- 연말: "새해 새집, 인테리어 견적 무료 비교"

### 상황별 소재
- 신혼: "신혼집 인테리어, 견적 비교로 시작"
- 이사: "이사 전 인테리어 견적 미리 비교"
- 구축: "구축 아파트 리모델링, 업체 비교"

> 소재를 변경할 때마다 심사가 필요하며 1~2 영업일 소요된다. 긴급 교체가 필요한 경우를 대비하여 예비 소재를 미리 승인받아 두는 것을 권장한다.
